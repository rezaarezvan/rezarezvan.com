Stability

A system is stable if and only if its characteristic equation has roots in the LHP, meaning that ℜ(s)<0\Re(s) < 0ℜ(s)<0.

For systems of order 2, this means that all the coefficients are >0> 0>0

For systems or order 3 or higher we use Routh–Hurwitz method.

Nyquists simplified criterion

To use the Nyquists simplified criterion, L(s)L(s)L(s) can not have a pole in the RHP. If this criterion is met, plot L(jω)L(j\omega)L(jω) for 0≤ω≤∞0 \leq \omega \leq \infty0≤ω≤∞.

If L(jω)L(j\omega)L(jω) passes through the negative real axis with (−1,0)(-1, 0)(−1,0) to its left, then the system is stable.
