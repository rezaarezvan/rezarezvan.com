Introduction

In this part we’ll cover so-called vector fields.

Definition

A vector field is a function, FFF:

F:D⊂R2→R2 ∣ 2D F:D⊂R3→R3 ∣ 3DF: D \subset \mathbb{R}^2 \rarr \mathbb{R}^2 \ | \ \text{2D} \\\ F: D \subset \mathbb{R}^3 \rarr \mathbb{R}^3 \ | \ \text{3D}F:D⊂R2→R2 ∣ 2D F:D⊂R3→R3 ∣ 3D

Gradient field

By this definiton, we can say that.

Let f:D⊂R2→Rf: D \subset \mathbb{R}^2 \rarr \mathbb{R}f:D⊂R2→R be a differentiable function. The gradient is ∇f(x,y)=⟨fx(x,y),fy(x,y)⟩\nabla f(x, y) = \langle f_x(x, y), f_y(x, y) \rangle∇f(x,y)=⟨fx​(x,y),fy​(x,y)⟩.

The gradient is therefore a vector field!

∇f:D→R2\nabla f: D \rarr \mathbb{R}^2∇f:D→R2

Which means, ∇f\nabla f∇f has larger (vector) length when the level curves are tighter together.

Definition

A vector field, FFF, is called conservative, if there is a function, fff, for which ∇f=F\nabla f = F∇f=F.

In this case, fff, is called a potential for FFF.

Integrating over vector fields

Suppose the vector field, FFF, represents some force on the plane/space and suppose there is a string, curved of the shape of a curve, CCC.

If a particle moves along the curve, the work done by FFF, to move it a distance:

∫CF⋅dr\int_C F \cdot dr∫C​F⋅dr

We call this the line integral of FFF over CCC.

Definition

If CCC is parameterized as r⃗(t)=⟨x(t),y(t),z(t)⟩\vec{r}(t) = \langle x(t), y(t), z(t) \rangler(t)=⟨x(t),y(t),z(t)⟩ in the interval, a≤t≤ba \leq t \leq ba≤t≤b. Then:

∫CF⋅dr=∫abF(x(t),y(t),z(t))⋅r′⃗(t) dt\int_C F \cdot dr = \int_a^b F(x(t), y(t), z(t)) \cdot \vec{r^\prime}(t)\ dt∫C​F⋅dr=∫ab​F(x(t),y(t),z(t))⋅r′(t) dt
