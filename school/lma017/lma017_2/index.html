Introduction

In this part we’ll define continuity when dealing with functions with several variables. We’ll also start looking at derivatives of functions of several variables.

Continuity

Let’s recall what continuity of a function with one variable is defined:

A function of one variable is continuous at a point, a∈Da \in Da∈D if:

lim⁡x→af(x)=f(a)\lim_{x \to a} f(x) = f(a)x→alim​f(x)=f(a)

In functions with several variables, it’s the same:

lim⁡(x,y)→(a,b)f(x,y)=f(a,b)\lim_{(x, y) \to (a, b)} f(x, y) = f(a, b)(x,y)→(a,b)lim​f(x,y)=f(a,b)

We say fff is continuous on its domain, if fff is continuous at each point, (a,b)∈D(a, b) \in D(a,b)∈D

Partial derivatives

Let’s first start with recalling the definition of derivative for functions with one variable:

lim⁡h→0f(x+h)−f(x)h\lim_{h \to 0} \dfrac{f(x + h) - f(x)}{h}h→0lim​hf(x+h)−f(x)​

Since we’re dealing with several variables, we need to choose what variable to respect, meaning what variable has a small increase.

Definition

The partial derivate of a function of two variables with respect to x, denoted by fxf_xfx​, is the function of two variables given by:

fx(x,y)=lim⁡h→0f(x+h,y)−f(x,y)hf_x(x, y) = \lim_{h \to 0} \dfrac{f(x + h, y) - f(x, y)}{h}fx​(x,y)=h→0lim​hf(x+h,y)−f(x,y)​

Similarly for yyy:

fy(x,y)=lim⁡h→0f(x,y+h)−f(x,y)hf_y(x, y) = \lim_{h \to 0} \dfrac{f(x, y + h) - f(x, y)}{h}fy​(x,y)=h→0lim​hf(x,y+h)−f(x,y)​

In other words what this means, the partial derivate with respect to a variable, we treat all other variables as they were constants.
