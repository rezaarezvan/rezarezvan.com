Part 8 - Lagrange multipliers

Date: 9/13/2023

Introduction

In this part we’ll cover the method of Lagrange multipliers. This is a strategy, for finding local minimum/maximums for a function with a constraint.

Let’s define it properly

Definition

Say we have a function, f(x,y)f(x, y)f(x,y), which we want to find the min/max points. But this function is subject of some constraint of the form, g(x,y)=kg(x, y) = kg(x,y)=k.

Recall our rectangular box problem from the last part:

V(x,y,z)=xyz2zx+2zy+yx=12V(x, y, z) = xyz \newline 2zx + 2zy + yx = 12V(x,y,z)=xyz2zx+2zy+yx=12

So if it’s difficult to try to rewrite this using only two variables, we can instead use this method.

Let’s first get a geometrical sense.

Geometrical sense of Lagrange multipliers

If min/max of fff, is subject to constraint, g(x,y)=kg(x, y) = kg(x,y)=k, which occurs at (x0,y0)(x_0, y_0)(x0​,y0​), then the level curve containing (x0,y0)(x_0, y_0)(x0​,y0​) must touch the constraint curve.

If two curves are touching at (x0,y0)(x_0, y_0)(x0​,y0​), in other words they have a common tangent line at (x0,y0)(x_0, y_0)(x0​,y0​).

Which is just equivalent of saying that they have a common perpendicular line at (x0,y0)(x_0, y_0)(x0​,y0​), and what do we know which is orthogonal to the level curve?

The gradient.

Therefore, this means:

∇f(x0,y0)∥∥∇g(x0,y0)\nabla f(x_0, y_0) \|\| \nabla g(x_0, y_0)∇f(x0​,y0​)∥∥∇g(x0​,y0​)

If two vectors are parallel, it’s equivalent of saying:

∇f(x0,y0)∥∥λ∇g(x0,y0)\boxed{\nabla f(x_0, y_0) \|\| \lambda \nabla g(x_0, y_0)}∇f(x0​,y0​)∥∥λ∇g(x0​,y0​)​

All of this assuming that ∇g≠0\nabla g \neq 0∇g=0

Let’s properly define our method now.

Lagrange multiplier method

To find minimum/maximum values of function, fff, subject to constraint, g(x,y)=kg(x, y) = kg(x,y)=k, assuming minimum/maximum points exists and ∇g≠0\nabla g \neq 0∇g=0

  1. Find all numbers x,yx, yx,y and λ\lambdaλ, such that:
∇f(x,y)=λ∇g(x,y)=k\nabla f(x, y) = \lambda \nabla g(x, y) = k∇f(x,y)=λ∇g(x,y)=k
  1. Compute values from step 1, choose the minimum/maximum value(s).

Since we have a unit circle, with equation, x2+y2=1x^2 + y^2 = 1x2+y2=1, we can view this as a constraint.

Remark

A little remark, it is important that the constraint has to be of the form g(x,y)=kg(x, y) = kg(x,y)=k. For example, our rectangle problem from last time, we can not describe a rectangular box in that form.

So no lagrange multiplier there. Let’s look at more difficult example.
