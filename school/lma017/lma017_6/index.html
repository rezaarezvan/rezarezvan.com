Part 6 - Maximum and minimum values

Date: 9/8/2023

Introduction

In this part we’ll cover how we find maximum and minimum values for a function, we’ll also see how this is related to the 2nd derivative.

Maximum and minimum values

Let’s first recall how we define maximum and minimum values for functions of one variable.

Definition

For maximum:

A function of one variable has local maximum at aaa, if f(a)≥f(x)f(a) \geq f(x)f(a)≥f(x) when xxx is near aaa.

For minimum: A function of one variable has local minimum at aaa, if f(a)≤f(x)f(a) \leq f(x)f(a)≤f(x) when xxx is near aaa.

Similarly, for two variables: A function of two variables has local maximum at (a,b)(a, b)(a,b) if f(a,b)≥f(x,y)f(a, b) \geq f(x, y)f(a,b)≥f(x,y) when (x,y)(x, y)(x,y) is near (a,b)(a, b)(a,b).

A function of two variables has local minimum at (a,b)(a, b)(a,b) if f(a,b)≤f(x,y)f(a, b) \leq f(x, y)f(a,b)≤f(x,y) when (x,y)(x, y)(x,y) is near (a,b)(a, b)(a,b).

For absolute max/min, we define them as: If f(a,b)≥f(x,y)f(a, b) \geq f(x, y)f(a,b)≥f(x,y) for any (x,y)∈D(x, y) \in D(x,y)∈D, we have an absolute maximum.

If f(a,b)≤f(x,y)f(a, b) \leq f(x, y)f(a,b)≤f(x,y) for any (x,y)∈D(x, y) \in D(x,y)∈D, we have an absolute minimum.

How do we find these minimums and maximums?

How to find min/max

Let’s recall for functions of one variable, min/max points have derivate of zero, therefore:

Proof

Let (a,b)(a, b)(a,b) be a point of local maximum and fx(a,b)f_x(a, b)fx​(a,b) and fy(a,b)f_y(a, b)fy​(a,b) exists. f(a,b)≥f(x,y)f(a, b) \geq f(x, y)f(a,b)≥f(x,y) when (x,y)(x, y)(x,y) near (a,b)(a, b)(a,b).

In particular, take y = b:

f(a,b)=f(x,b) ∣ x near af(a, b) = f(x, b) \ | \ x \text{ near } af(a,b)=f(x,b) ∣ x near a

Let g(x)=f(x,b)g(x) = f(x, b)g(x)=f(x,b), this means:

g(a)=f(a,b)g(a) = f(a, b)g(a)=f(a,b)

We can say that aaa is a point of local maximum:

g′(a)=0g'(a) = 0g′(a)=0

Which is just the same as:

fx(a,b)=0f_x(a, b) = 0fx​(a,b)=0

Similarly, for fy(a,b)=0f_y(a, b) = 0fy​(a,b)=0, we just choose to fix x=ax = ax=a instead.

Geometrical sense

In functions of one variable, at points of local min/max, the tangent line is horizontal to our point, if the line exists.

For functions of two variables, instead of a line, we have a plane. The tangent plane is horizontal to our point, if the plane exists.

Definition

Let’s properly define what we’ve covered so far:

(a,b)(a, b)(a,b) is a critical point if ∇f(a,b)=0 or ∇f(a,b) ∄\nabla f(a, b) = 0 \textbf{ or } \nabla f(a, b) \ \nexists∇f(a,b)=0 or ∇f(a,b) ∄

In other words, if (a,b)(a, b)(a,b) is a critical point of local minimum/maximum then it is a critical point.

However, the other way around this implication is not true. A critical point might not be a local minimum/maxium.

Example

Find the critical point(s) for the function:

f(x,y)=x2+y2−2x−6y+14f(x, y) = x^2 + y^2 - 2x - 6y + 14f(x,y)=x2+y2−2x−6y+14

We find that x=1x = 1x=1 and y=3y = 3y=3. This means (1,3)(1, 3)(1,3) is a critical point (and the only one!).

But how do we actually know if a critical point is a local minimum or maximum?

2nd derivate test

For functions of one variable, if we do the 2nd derivate test:

Suppose aaa is a point and f′′(a)f''(a)f′′(a) exists, then:

f′′(a)>0 ∣ local minf′′(a)<0 ∣ local maxf′′(a)=0 ∣ does not say anythingf''(a) > 0 \ | \ \text{local min} \newline f''(a) < 0 \ | \ \text{local max} \newline f''(a) = 0 \ | \ \text{does not say anything}f′′(a)>0 ∣ local minf′′(a)<0 ∣ local maxf′′(a)=0 ∣ does not say anything
