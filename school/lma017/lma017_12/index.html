Part 12 - Change of variables

Date: 9/22/2023

Introduction

In this part we’ll cover how to perform change of variables for double and triple integrals, as well as the idea and power with polar coordinates.

Change of variable

For functions of one variable:

∫abf(g(x))g′(x) dx\int_a^b f(g(x))g'(x)\ dx∫ab​f(g(x))g′(x) dx

Let u=g(x)u = g(x)u=g(x) and therefore du=g′(x) dxdu = g'(x)\ dxdu=g′(x) dx.

Then:

∫g(a)g(b)f(u) du\int_{g(a)}^{g(b)} f(u)\ du∫g(a)g(b)​f(u) du

But in two variables, this is a bit tricker, we’ll have to use polar coordinates.

Polar coordinates

Instead of describing a point as (x,y)(x, y)(x,y) we can describe it using the magnitude (from the origin) and a angle.

r≥00≤θ≤2πr \geq 0 \newline 0 \leq \theta \leq 2\pir≥00≤θ≤2π

Then:

x=rcos(θ)y=rsin(θ)x = r cos(\theta) \newline y = r sin(\theta)x=rcos(θ)y=rsin(θ)

When to change to polar coordinates

  1. If fff looks nicer in polar coordinates

and

  1. DDD looks nice in polar coordinates

Example

Say we have the unit circle:

D=(x,y)∣x2+y2≤1D = \\{(x, y) | x^2 + y^2 \leq 1\\}D=(x,y)∣x2+y2≤1

In polar coordinates:

S=(r,θ)∣0≤r≤1,0≤θ≤2πS = \\{(r, \theta) | 0 \leq r \leq 1, 0 \leq \theta \leq 2\pi \\}S=(r,θ)∣0≤r≤1,0≤θ≤2π

Which is just a rectangle! Which we know how to integrate over!

D=(x,y)∣1≤x2+y2≤4D = \\{(x, y) | 1 \leq x^2 + y^2 \leq 4\\}D=(x,y)∣1≤x2+y2≤4

In polar coordinates:

S=(r,θ)∣1≤r≤2,0≤θ≤2πS = \\{(r, \theta) | 1 \leq r \leq 2, 0 \leq \theta \leq 2\pi \\}S=(r,θ)∣1≤r≤2,0≤θ≤2π

Or something even more complex:

S=(r,θ)∣0≤r≤1,θ1≤θ≤θ2S = \\{(r, \theta) | 0 \leq r \leq 1, \theta_1 \leq \theta \leq \theta_2 \\}S=(r,θ)∣0≤r≤1,θ1​≤θ≤θ2​

Naive guess

But, how do we change to polar coordinates?

Let’s take a naive guess:

∬Df(x,y) dA=∬Sf(rcos(θ),rsin(θ)) dA\iint_D f(x, y)\ dA = \iint_S f(r cos(\theta), r sin(\theta))\ dA∬D​f(x,y) dA=∬S​f(rcos(θ),rsin(θ)) dA

Let’s try!

Given f(x,y)=1f(x, y) = 1f(x,y)=1 and D=(x,y)∣x2+y2≤1D = \\{(x,y) | x^2 + y^2 \leq 1\\}D=(x,y)∣x2+y2≤1.

∬D1 dA=Area of D=πr2=π\iint_D 1\ dA = \text{Area of D} = \pi r^2 = \pi∬D​1 dA=Area of D=πr2=π

Polar:

S=(r,θ)∣0≤r≤1,0≤θ≤2πS = \\{(r, \theta) | 0 \leq r \leq 1, 0 \leq \theta \leq 2\pi \\}S=(r,θ)∣0≤r≤1,0≤θ≤2π ∬S1 dA=∫02π∫01r dr dθ=…=π\iint_S 1\ dA = \int_0^{2\pi} \int_0^1 r\ dr\ d\theta = \ldots = \pi∬S​1 dA=∫02π​∫01​r dr dθ=…=π

So this doesn’t yield the same result.

Definition

∬Sg(r,θ) dA=∬Sf(rcos(θ),rsin(θ))⋅r dA\iint_S g(r, \theta)\ dA = \iint_S f(r cos(\theta), r sin(\theta)) \cdot r\ dA∬S​g(r,θ) dA=∬S​f(rcos(θ),rsin(θ))⋅r dA
