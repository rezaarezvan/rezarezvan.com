Thermodynamics and Theory of Heat

Effiency

Let’s start with something which is quite simple to understand. The famous quote that “Energy is neither created nor destroyed, only changed from one form to another”. With this in mind we understand that we never ‘create’ energy. It’s also a sensible conclusion that if we had machines with 100% effiency, we wouldn’t have whole fields in physics about energy-effiency.

Informal defintion of effiency

(We will later define a more formal definiton of effiency) $$ e - [\text{Dimension less}], 0 \leq e \leq 1 \newline e = \frac{\text{Useful energy}}{\text{Total input energy}} $$

Real Life Examples

A usual car that runs on gas has a effiency of $\approx$ 20-25% (e $\approx$ 0.2 - 0.25)

Thermal expansion

Imagine we have a block of some kind of material. If we apply heat to this block it’s natural that it will expand. The same logic goes if we lower the heat/tempurate - it will compress instead.

Formula for Thermal Expansion

$$ L = L_0(1 + \alpha \Delta T) \text{, where:} \newline \alpha - \text{Expansion coefficent } [K^{-1}] $$

This is for a 1D expansion, if the expansion happens in 2D it is:

$$ L = L_0(1 + \beta \Delta T) \text{, where:} \newline \beta - \text{Expansion coefficent } [K^{-1}] \newline \text{Note: } \beta \approx 2\alpha $$

Thermal Conductivity

Say if we have a system of 2 bodies, one which gains energy/heats up from an external source, with a opening to the second body.

This will transfer the heat from the hot body, $T_h$ to the colder body $T_c$ since some heat will be lost in the transfer. The opening has a so called thermal conductivity, the ability to transfer heat. With this in mind we can derive.

$$ P = \frac{dQ}{dt} = Ak \frac{T_h - T_c}{l} \text{, where:} \newline P - \text{Effect } [W] \newline Q - \text{Heat exchange } [J] \newline A - \text{Cross-sectional Area} \newline k - \text{Thermal Conductivity } [\frac{W}{mK}] \newline l - \text{Length of the opening} $$

We can rewrite this also as: $$ P = -Ak \frac{dT}{dx} \text{, or more general} \newline P = -Ak \nabla T $$
