So I decided to make a Vim-like text editor. I took a lot of inspiration from this post from medium, so all credits to Kofi.

However I wanted to it more Vim-like since it would be fun. So I did it. I learnt a lot of ‘Think-in-a-Rust-way’. Which is refreshing.
