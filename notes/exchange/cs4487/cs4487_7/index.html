Expectation Maximization (EM)

EM solves a maximum likelihood problem of the form,

L(θ)=∑i=1Mlog⁡p(x(i);θ)=∑i=1Mlog⁡∑z(i)=1Kp(x(i),z(i);θ)L(\mathbf{\theta}) = \sum_{i = 1}^M \log p(\mathbf{x}^{(i)}; \mathbf{\theta}) = \sum_{i = 1}^M \log \sum_{\mathbf{z}^{(i)} = 1}^K p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})L(θ)=i=1∑M​logp(x(i);θ)=i=1∑M​logz(i)=1∑K​p(x(i),z(i);θ)
  • θ\mathbf{\theta}θ: Parameters of the probalistic model we are trying to find.
  • {x(i)}i=1M\{x^{(i)}\}_{i = 1}^M{x(i)}i=1M​: Observed training examples.
  • {z(i)}i=1M\{z^{(i)}\}_{i = 1}^M{z(i)}i=1M​: Unobserved latent variables. (e.g., in GMM, z(i)z^{(i)}z(i) indicates which one of the KKK clusters x(i)x^{(i)}x(i) belongs to, which is unobserved.)

Jensen’s Inequality

Suppose f:R↦Rf : \mathbb{R} \mapsto \mathbb{R}f:R↦R is concave, then for all probability distributions ppp, we have,

f(E[x])≥E[f(x)].f(\mathbb{E} [\mathbf{x}]) \geq \mathbb{E} [f(\mathbf{x})].f(E[x])≥E[f(x)].

Where the expectation is taken with respect to the random variable x\mathbf{x}x drawn from the probability distribution ppp.

The equality holds if and only if

  1. x\mathbf{x}x is a constant, or,
  2. fff is an affine function (i.e., f(x)=aTx+bf(\mathbf{x}) = a^T \mathbf{x} + bf(x)=aTx+b).

EM Derivation

Let’s derive the EM algorithm for the maximum likelihood problem.

L(θ)=∑i=1Mlog⁡p(x(i);θ)=∑i=1Mlog⁡∑z(i)=1Kp(x(i),z(i);θ)=∑i=1Mlog⁡∑z(i)=1Kq(z(i))p(x(i),z(i);θ)q(z(i))=∑i=1Mlog⁡Ez(i)∼q[p(x(i),z(i);θ)q(z(i))]\begin{aligned} L(\mathbf{\theta}) &= \sum_{i = 1}^M \log p(\mathbf{x}^{(i)}; \mathbf{\theta}) \newline &= \sum_{i = 1}^M \log \sum_{\mathbf{z}^{(i)} = 1}^K p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta}) \newline &= \sum_{i = 1}^M \log \sum_{\mathbf{z}^{(i)} = 1}^K q(\mathbf{z}^{(i)}) \frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})} \newline &= \sum_{i = 1}^M \log \mathbb{E}_{\mathbf{z}^{(i)} \sim q} \left[ \frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})} \right] \newline \end{aligned}L(θ)​=i=1∑M​logp(x(i);θ)=i=1∑M​logz(i)=1∑K​p(x(i),z(i);θ)=i=1∑M​logz(i)=1∑K​q(z(i))q(z(i))p(x(i),z(i);θ)​=i=1∑M​logEz(i)∼q​[q(z(i))p(x(i),z(i);θ)​]​

Now we can apply Jensen’s inequality to the above equation,

L(θ)≥∑i=1MEz(i)∼q[log⁡p(x(i),z(i);θ)q(z(i))]\begin{aligned} L(\mathbf{\theta}) &\geq \sum_{i = 1}^M \mathbb{E}_{\mathbf{z}^{(i)} \sim q} \left[ \log \frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})} \right] \newline \end{aligned}L(θ)​≥i=1∑M​Ez(i)∼q​[logq(z(i))p(x(i),z(i);θ)​]​

Let’s go back to sum notation,

L(θ)=∑i=1M∑z(i)=1Kq(z(i))log⁡p(x(i),z(i);θ)q(z(i))=∑i=1M∑z(i)=1Kq(z(i))log⁡p(x(i),z(i);θ)−∑i=1M∑z(i)=1Kq(z(i))log⁡q(z(i))\begin{aligned} L(\mathbf{\theta}) &= \sum_{i = 1}^M \sum_{\mathbf{z}^{(i)} = 1}^K q(\mathbf{z}^{(i)}) \log \frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})} \newline &= \sum_{i = 1}^M \sum_{\mathbf{z}^{(i)} = 1}^K q(\mathbf{z}^{(i)}) \log p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta}) \newline &- \sum_{i = 1}^M \sum_{\mathbf{z}^{(i)} = 1}^K q(\mathbf{z}^{(i)}) \log q(\mathbf{z}^{(i)}) \newline \end{aligned}L(θ)​=i=1∑M​z(i)=1∑K​q(z(i))logq(z(i))p(x(i),z(i);θ)​=i=1∑M​z(i)=1∑K​q(z(i))logp(x(i),z(i);θ)−i=1∑M​z(i)=1∑K​q(z(i))logq(z(i))​

Let’s call this last expression for ell(θ)ell(\mathbf{\theta})ell(θ), this is a lower bound of the original objective L(θ)L(\mathbf{\theta})L(θ). The equality holds when p(x(i),z(i);θ)q(z(i))\frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})}q(z(i))p(x(i),z(i);θ)​ is a constant.

This can be achieved for q(z(i))=p(z(i)∣x(i);θ)q(z^{(i)}) = p(z^{(i)} | x^{(i)}; \theta)q(z(i))=p(z(i)∣x(i);θ).

The EM algorithm aims to optimize the lower bound ell(θ)ell(\mathbf{\theta})ell(θ),

θ⋆=arg⁡max⁡θℓ(θ)=arg⁡max⁡θ∑i=1M∑z(i)=1Kq(z(i))log⁡p(x(i),z(i);θ)q(z(i))\mathbf{\theta}^{\star} = \underset{\mathbf{\theta}}{\arg \max} \ell(\mathbf{\theta}) = \underset{\mathbf{\theta}}{\arg \max} \sum_{i = 1}^M \sum_{\mathbf{z}^{(i)} = 1}^K q(\mathbf{z}^{(i)}) \log \frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})}θ⋆=θargmax​ℓ(θ)=θargmax​i=1∑M​z(i)=1∑K​q(z(i))logq(z(i))p(x(i),z(i);θ)​

EM repeatedly performs the following two steps until convergence.

At ttt-th iteration,

  1. E-step: For each index iii, we compute,
q(t)(z(i))=p(z(i)∣x(i);θ(t))q^{(t)}(z^{(i)}) = p(z^{(i)} | x^{(i)}; \mathbf{\theta}^{(t)})q(t)(z(i))=p(z(i)∣x(i);θ(t))
  1. M-step: Compute,
θ(t+1)=arg⁡max⁡θ∑i=1M∑z(i)=1q(t)(z(i))log⁡p(x(i),z(i);θ)\mathbf{\theta}^{(t + 1)} = \underset{\mathbf{\theta}}{\arg \max} \sum_{i = 1}^M \sum_{\mathbf{z}^{(i)} = 1} q^{(t)}(\mathbf{z}^{(i)}) \log p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})θ(t+1)=θargmax​i=1∑M​z(i)=1∑​q(t)(z(i))logp(x(i),z(i);θ)

In the E-step, we do not fill in the unobserved z(i)z^{(i)}z(i) with hard values, but find a posterior distribution q(z(i))q(z^{(i)})q(z(i)), given x(i)x^{(i)}x(i) and θ(t)\theta^{(t)}θ(t), i.e.,

q(t)(z(i))=p(z(i)∣x(i);θ(t)).q^{(t)}(z^{(i)}) = p(z^{(i)} | x^{(i)}; \mathbf{\theta}^{(t)}).q(t)(z(i))=p(z(i)∣x(i);θ(t)).

In the M-step, we maximize the lower bound ell(θ)ell(\mathbf{\theta})ell(θ), while holding q(t)(z(i))q^{(t)}(z^{(i)})q(t)(z(i)) fixed, which is computed from the E-step.

The M-step optimization can be done efficiently in most cases. For example, in GMM, we have a closed-form solution for all parameters.

EM Convergence

Assuming θ(t)\mathbf{\theta}^{(t)}θ(t) and θ(t+1)\mathbf{\theta}^{(t + 1)}θ(t+1) are the parameters from two successive iterations of EM, we have,

  1. By definition, this is the (log) likelihood of the data.
  2. By marginalization over z(i)z^{(i)}z(i) and multiplication an arbitrary distribution q(z(i))q(z^{(i)})q(z(i)) to both numerator and denominator inside log.
  3. By Jensen’s inequality where equality condition is satisfied by setting q(t)(z(i))=p(z(i)∣x(i);θ(t))q^{(t)}(z^{(i)}) = p(z^{(i)} | x^{(i)}; \mathbf{\theta}^{(t)})q(t)(z(i))=p(z(i)∣x(i);θ(t)).
  4. By M-step of EM, where we maximize (3), holding q(t)(z(i))q^{(t)}(z^{(i)})q(t)(z(i)) fixed.
  5. By Jensen’s inequality (in reverse order). Note that we have already updated θ\mathbf{\theta}θ from θ(t)\mathbf{\theta}^{(t)}θ(t) to θ(t+1)\mathbf{\theta}^{(t + 1)}θ(t+1), q(t)(z(i))q^{(t)}(z^{(i)})q(t)(z(i)) may now not satisfy the equality condition.
  6. By definition, this is the lower bound of the likelihood.

Hence, EM causes the likelihood to increase monotonically.

Remark

One remark that we have to do is, if we define the EM as,

J(q,θ)=∑i=1M∑z(i)=1Kq(z(i))log⁡p(x(i),z(i);θ)q(z(i))J(q, \mathbf{\theta}) = \sum_{i = 1}^M \sum_{\mathbf{z}^{(i)} = 1}^K q(\mathbf{z}^{(i)}) \log \frac{p(\mathbf{x}^{(i)}, \mathbf{z}^{(i)}; \mathbf{\theta})}{q(\mathbf{z}^{(i)})}J(q,θ)=i=1∑M​z(i)=1∑K​q(z(i))logq(z(i))p(x(i),z(i);θ)​

We can view this as coordinate ascent on JJJ, in which the E-step maximizes JJJ with respect to qqq, and the M-step maximizes JJJ with respect to θ\mathbf{\theta}θ. One can easily prove this using Lagrangian multipliers!

Clustering Summary

  • Clustering Task
    • Given a set of input vectors D={x(i)}i=1M\mathcal{D} = \{x^{(i)}\}_{i = 1}^MD={x(i)}i=1M​, with x(i)∈RNx^{(i)} \in \mathbb{R}^Nx(i)∈RN, group similar x(i)x^{(i)}x(i) into clusters.
      • Estimate a cluster center, representing the data points in that cluster.
      • Predict the cluster for a new data point.
  • Exhaustive clustering
    • Cluster shape: Arbitrary shape.
    • Principle: Minimize an assumed clustering criterion with brute-force search.
    • Pros: Optimal under the given clustering criterion.
    • Cons: Impractical to construct the clustering criterion, prohibitive to compute.
  • KKK-means
    • Cluster shape: Circular.
    • Principle: Minimize distance to cluster center.
    • Pros: Simple and scalable (MiniBatchKMeans).
    • Cons: Sensitive to initialization, could get bad solutions due to local minima, need to choose KKK.
  • Gaussian Mixture Model (GMM)
    • Cluster shape: Elliptical.
    • Principle: Maximum likelihood using expectation maximization.
    • Pros: Elliptical cluster shapes.
    • Cons: Sensitive to initialization, could get bad solutions due to local minima, need to choose KKK.
  • Feature normalization
    • Feature normalization is typically required for clustering.
    • E.g., algorithms based on Eucledian distance (KKK-means).

Dimensionality Reduction

Transform high-dimensional vectors into low-dimensional vectors. Dimensions in the low-dimensional data represent co-occuring features in the high-dimensional data. Dimensions in the low-dimensional data may have semantic meaning.

For example, in document analysis.

  • High-dimensional data: Bag-of-word vectors of documents.
  • Low-dimensional data: Each dimension represents similarity to a topic.

Reasons for Dimensionality Reduction

  • Preprocessing, makes the dataset easier to use.
  • Reduce computational cost of running machine learning algorithms.
  • Can be used to “de-noise” data by projecting to lower-dimensional space and then projecting back to the original high-dimensional space.
  • Makes the results easier to understand (e.g., visualization).

Dimensionality Reduction VS. Feature Selection

The goal of feature selection is to remove features that are not informative with respect to the class label. This obviously reduces the dimensionality of the feature space.

Dimensionality reduction can be used to find a meaningful lower-dimensional feature space even when there is information in each feature dimension so that none can be discarded.

Another important property of dimensionality reduction is that it is unsupervised.

While dimensionality reduction can be seen as a simplistic form of (data) compression, it is not equivalent to it, as the goal of compression is to reduce the expected code length (which is lower bounded by entropy) of the representation not only the dimensionality.

For example, in lossless compression, arithmetic coding encodes the entire data into a single number, an arbitrary-precision fraction qqq where 0.0≤q<1.00.0 \leq q < 1.00.0≤q<1.0.

Linear Dimensionality Reduction

In linear dimensionality reduction we project the original data onto a lower-dimensional hyperplane (e.g., line, plane).

I.e., move and rotate the coordinate axis of the data, then we represent the data with coordinates in the new component space.

Mathematically, this can be written as,

x(i)=∑k=1Kzk(i)bk\mathbf{x}^{(i)} = \sum_{k = 1}^K z_k^{(i)} \mathbf{b}_kx(i)=k=1∑K​zk(i)​bk​

where bk\mathbf{b}_kbk​ is a basis vector and zk(i)∈Rz_k^{(i)} \in \mathbb{R}zk(i)​∈R is the corresponding weight.

Connection to Linear Regression

If we focus on the jjj-th entry of x(i)\mathbf{x}^{(i)}x(i), we have,

xj(i)=∑k=1Kzk(i)bjkx_j^{(i)} = \sum_{k = 1}^K z_k^{(i)} b_{jk}xj(i)​=k=1∑K​zk(i)​bjk​

This expression can be seen as linear regression.

  • xj(i)x_j^{(i)}xj(i)​ is the target.
  • zk(i)z_k^{(i)}zk(i)​ for each kkk are the weights.
  • bjkb_{jk}bjk​ for each kkk are the features.

Alternatively, we may view zk(i)z_k^{(i)}zk(i)​ as feature and bjkb_{jk}bjk​ as weights.

Unlike linear regression, we only know “targets”. We must learn both features and weights.

Learning Criterion

The learning problem for linear dimensionality reduction is to estimate values for both Z\mathbf{Z}Z and B\mathbf{B}B given only the noisy observations of X\mathbf{X}X.

One possible learning criterion is to minimize the sum of squared errors when reconstructing X\mathbf{X}X from Z\mathbf{Z}Z and B\mathbf{B}B.

This leads to,

arg⁡min⁡Z,B∥X−ZB∥F2\underset{\mathbf{Z}, \mathbf{B}}{\arg \min} \Vert \mathbf{X} - \mathbf{ZB} \Vert_F^2Z,Bargmin​∥X−ZB∥F2​

Where ∥⋅∥F\Vert \cdot \Vert_F∥⋅∥F​ is the Frobenius norm of a matrix, defined as,

∥A∥F=∑ijAij2\Vert \mathbf{A} \Vert_F = \sqrt{\sum_{ij} A_{ij}^2}∥A∥F​=ij∑​Aij2​​

Alternating Least Squares

By leveraging the OLS solution for linear regression, we can estimate Z\mathbf{Z}Z and B\mathbf{B}B using Alternating Least Squares (ALS).

Starting from some random initialization, ALS iterates between two steps until covergence.

  1. Assume Z\mathbf{Z}Z is fixed and optimize B\mathbf{B}B,
B←(ZTZ)−1ZTX\mathbf{B} \leftarrow (\mathbf{Z}^T \mathbf{Z})^{-1} \mathbf{Z}^T \mathbf{X}B←(ZTZ)−1ZTX
  1. Assume B\mathbf{B}B is fixed and optimize Z\mathbf{Z}Z,
ZT←(BBT)−1BXT\mathbf{Z}^T \leftarrow (\mathbf{B} \mathbf{B}^T)^{-1} \mathbf{B} \mathbf{X}^TZT←(BBT)−1BXT

Lack of Uniqueness for Optimal Parameters

Suppose we run the ALS algorithm to convergence and obtain optimal parameters Z⋆\mathbf{Z}^{\star}Z⋆ and B⋆\mathbf{B}^{\star}B⋆ such that,

ℓ⋆=∥X−Z⋆B⋆∥F2\ell^{\star} = \Vert \mathbf{X} - \mathbf{Z}^{\star} \mathbf{B}^{\star} \Vert_F^2ℓ⋆=∥X−Z⋆B⋆∥F2​

Let R∈RK×K\mathbf{R} \in \mathbb{R}^{K \times K}R∈RK×K be an arbitrary invertible matrix.

A K×KK \times KK×K matrix R\mathbf{R}R is invertible, if there exists a K×KK \times KK×K matrix S\mathbf{S}S such that RS=SR=I\mathbf{R} \mathbf{S} = \mathbf{S} \mathbf{R} = \mathbf{I}RS=SR=I. Which we also can denote as R−1=S\mathbf{R}^{-1} = \mathbf{S}R−1=S.

We obtain a different set of parameters Z~=Z⋆R\mathbf{\tilde{Z}} = \mathbf{Z}^{\star} \mathbf{R}Z~=Z⋆R and B~=R−1B⋆\mathbf{\tilde{B}} = \mathbf{R}^{-1} \mathbf{B}^{\star}B~=R−1B⋆, with the same optimal value,

ℓ⋆=∥X−Z⋆(I)B⋆∥F2=∥X−Z⋆(RR−1)B⋆∥F2=∥X−Z~B~∥F2\ell^{\star} = \Vert \mathbf{X} - \mathbf{Z}^{\star} (\mathbf{I}) \mathbf{B}^{\star} \Vert_F^2 = \Vert \mathbf{X} - \mathbf{Z}^{\star} (\mathbf{R} \mathbf{R}^{-1}) \mathbf{B}^{\star} \Vert_F^2 = \Vert \mathbf{X} - \mathbf{\tilde{Z}} \mathbf{\tilde{B}} \Vert_F^2ℓ⋆=∥X−Z⋆(I)B⋆∥F2​=∥X−Z⋆(RR−1)B⋆∥F2​=∥X−Z~B~∥F2​

We can obtain the global optimal solution(s) and make them unique by specifying additional criteria.

Singular Value Decomposition (SVD)

Let X\mathbf{X}X be a M×NM \times NM×N matrix, with M≥NM \geq NM≥N. It can be facotrized as,

X=U(Σ0)VT\mathbf{X} = \mathbf{U} \begin{pmatrix} \mathbf{\Sigma} \newline \mathbf{0} \end{pmatrix} \mathbf{V}^TX=U(Σ0​)VT

Where U∈RM×M\mathbf{U} \in \mathbb{R}^{M \times M}U∈RM×M and V∈RN×N\mathbf{V} \in \mathbb{R}^{N \times N}V∈RN×N are orthogonal, i.e.,

UTU=UUT=IM,VTV=VVT=IN\mathbf{U}^T \mathbf{U} = \mathbf{U} \mathbf{U}^T = \mathbf{I}_M, \quad \mathbf{V}^T \mathbf{V} = \mathbf{V} \mathbf{V}^T = \mathbf{I}_NUTU=UUT=IM​,VTV=VVT=IN​

Columns of U\mathbf{U}U and V\mathbf{V}V are called left and right singular vectors of X\mathbf{X}X, respectively.

Σ∈RM×N\mathbf{\Sigma} \in \mathbb{R}^{M \times N}Σ∈RM×N is a diagonal matrix,

Eckart-Young-Mirsky Theorem

Given a M×NM \times NM×N matrix X\mathbf{X}X of rank R≤min⁡{M,N}R \leq \min \{M, N\}R≤min{M,N} and its singular value decomposition X=UΣRVT\mathbf{X} = \mathbf{U} \mathbf{\Sigma}_R \mathbf{V}^TX=UΣR​VT with singular values σ1≥σ2≥…≥σR>0\sigma_1 \geq \sigma_2 \geq \ldots \geq \sigma_R > 0σ1​≥σ2​≥…≥σR​>0 and the rest being zero, then among all M×NM \times NM×N matrices of lower rank K≤RK \leq RK≤R, the best approximation is Y⋆=UΣKVT\mathbf{Y}^{\star} = \mathbf{U} \mathbf{\Sigma}_K \mathbf{V}^TY⋆=UΣK​VT, where ΣK\mathbf{\Sigma}_KΣK​ is the diagonal matrix with singular values σ1,σ2,…,σK\sigma_1, \sigma_2, \ldots, \sigma_Kσ1​,σ2​,…,σK​ in the sense that,

∥X−Y⋆∥F2=min⁡{∥X−Y∥F2;Y∈RM×N,rankY≤K}\Vert \mathbf{X} - \mathbf{Y}^{\star} \Vert_F^2 = \min \{\Vert \mathbf{X} - \mathbf{Y} \Vert_F^2; \mathbf{Y} \in \mathbb{R}^{M \times N}, \text{rank} \mathbf{Y} \leq K\}∥X−Y⋆∥F2​=min{∥X−Y∥F2​;Y∈RM×N,rankY≤K}
