Eigenvectors

Let’s do a quick linear algebra recap.

Assume A∈RN×N,v∈CN×1\mathbf{A} \in \mathbb{R}^{N \times N}, \mathbf{v} \in \mathbb{C}^{N \times 1}A∈RN×N,v∈CN×1, and λ∈C\lambda \in \mathbb{C}λ∈C.

If Av=λv\mathbf{A} \mathbf{v} = \lambda \mathbf{v}Av=λv, then v\mathbf{v}v is a right eigenvector of A\mathbf{A}A with eigenvalue λ\lambdaλ.

If ATv=λv\mathbf{A}^T \mathbf{v} = \lambda \mathbf{v}ATv=λv, then v\mathbf{v}v is a left eigenvector of A\mathbf{A}A with eigenvalue λ\lambdaλ. (Equivalently, vTA=λvT\mathbf{v}^T \mathbf{A} = \lambda \mathbf{v}^TvTA=λvT.)

If A\mathbf{A}A is symmetric such that A=AT\mathbf{A} = \mathbf{A}^TA=AT, then the left and right eigenvectors of A\mathbf{A}A are the same with the same eigenvalues.

Linear Independence

Linear independence is arguably the most important concept in linear algebra.

A set of vectors {v1,v2,…,vN}\{\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_N\}{v1​,v2​,…,vN​} is linearly independent if vector equation,

a1v1+a2v2+…+aNvN=0,a_1 \mathbf{v}_1 + a_2 \mathbf{v}_2 + \ldots + a_N \mathbf{v}_N = \mathbf{0},a1​v1​+a2​v2​+…+aN​vN​=0,

has only the trivial solution a1=a2=…=aN=0a_1 = a_2 = \ldots = a_N = 0a1​=a2​=…=aN​=0.

{v1,v2,…,vN}\{\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_N\}{v1​,v2​,…,vN​} is linearly dependent if there exists numbers a1,a2,…,aNa_1, a_2, \ldots, a_Na1​,a2​,…,aN​ not all equal to zero, such that,

a1v1+a2v2+…+aNvN=0.a_1 \mathbf{v}_1 + a_2 \mathbf{v}_2 + \ldots + a_N \mathbf{v}_N = \mathbf{0}.a1​v1​+a2​v2​+…+aN​vN​=0.

Assuming a1≠0a_1 \neq 0a1​=0, we have v1=−a2a1v2−…−aNa1vN\mathbf{v}_1 = -\frac{a_2}{a_1} \mathbf{v}_2 - \ldots - \frac{a_N}{a_1} \mathbf{v}_Nv1​=−a1​a2​​v2​−…−a1​aN​​vN​.

Some Matrices

An N×NN \times NN×N square matrix A\mathbf{A}A is invertible if there exists an N×NN \times NN×N square matrix B\mathbf{B}B such that,

AB=BA=IN,\mathbf{A} \mathbf{B} = \mathbf{B} \mathbf{A} = \mathbf{I}_N,AB=BA=IN​,

Equivalently, the columns/rows of A\mathbf{A}A are linearly independent.

A square matrix Q\mathbf{Q}Q is orthogonal if its columns and rows are orthogonal unit vectors (orthonormal vectors),

QQT=QTQ=I.\mathbf{Q} \mathbf{Q}^T = \mathbf{Q}^T \mathbf{Q} = \mathbf{I}.QQT=QTQ=I.

A square matrix A\mathbf{A}A is diagonalizable if there exists an invertible matrix P\mathbf{P}P and a diagonal matrix D\mathbf{D}D such that,

P−1AP=D,\mathbf{P}^{-1} \mathbf{A} \mathbf{P} = \mathbf{D},P−1AP=D,

or equivalently,

A=PDP−1.\mathbf{A} = \mathbf{P} \mathbf{D} \mathbf{P}^{-1}.A=PDP−1.

Real symmetric matrices are diagonalizable by orthogonal matrices. This can be proven by the spectral theorem.

Principal Component Analysis

Principal Component Analysis (PCA) is an unsupervised method, given a data matrix X∈RM×N\mathbf{X} \in \mathbb{R}^{M \times N}X∈RM×N, the goal of PCA is to identify the directions of maximum variance contained in the data.

We need to choose the basis vectors along the maximum variance (longest extent) of the data. The basis vectors are called principal components (PC).

Sample Variance in a Given Direction

Let v∈RN\mathbf{v} \in \mathbb{R}^Nv∈RN such that ∥v∥2=vTv=1\Vert \mathbf{v} \Vert_2 = \mathbf{v}^T \mathbf{v} = 1∥v∥2​=vTv=1.

The variance in the direction v\mathbf{v}v is given by the expression,

1M∑i=1M(vTx(i)−μ)2,\frac{1}{M} \sum_{i = 1}^M (\mathbf{v}^T \mathbf{x}^{(i)} - \mathbf{\mu})^2,M1​i=1∑M​(vTx(i)−μ)2,

where μ=1M∑i=1MvTx(i)\mathbf{\mu} = \frac{1}{M} \sum_{i = 1}^M \mathbf{v}^T \mathbf{x}^{(i)}μ=M1​∑i=1M​vTx(i) is the mean of the projected data.

The Direction of Maximum Variance

Suppose we want to identify the direction v1\mathbf{v}_1v1​ of maximum variance given the data matrix X\mathbf{X}X.

We can formulate this optimization problem as follows,

max⁡v1MvTXTXvsubject to∥v∥2=1.\begin{align*} \underset{\mathbf{v}}{\max} & \quad \frac{1}{M} \mathbf{v}^T \mathbf{X}^T \mathbf{X} \mathbf{v} \newline \text{subject to} & \quad \Vert \mathbf{v} \Vert_2 = 1. \end{align*}vmax​subject to​M1​vTXTXv∥v∥2​=1.​

KKK Largest Directions of Variance

Suppose instead of just the direction of maximum variance, we want the KKK largest directions of variance that are all mutually orthogonal.

Finding the second-largest direction of variance corresponds to solving the problem,

max⁡v1MvTXTXvsubject to∥v∥2=1,vTv1=0.\begin{align*} \underset{\mathbf{v}}{\max} & \quad \frac{1}{M} \mathbf{v}^T \mathbf{X}^T \mathbf{X} \mathbf{v} \newline \text{subject to} & \quad \Vert \mathbf{v} \Vert_2 = 1, \newline & \quad \mathbf{v}^T \mathbf{v}_1 = 0. \end{align*}vmax​subject to​M1​vTXTXv∥v∥2​=1,vTv1​=0.​

In general, the top KKK directions of variance v1,…,vK\mathbf{v}_1, \ldots, \mathbf{v}_Kv1​,…,vK​ are given by the KKK eigenvectors corresponding to the KKK largest eigenvalues of 1MXTX\frac{1}{M} \mathbf{X}^T \mathbf{X}M1​XTX.

Dimensionality Reduction with PCA

The informal algorithm can be described as follows,

  1. Subtract the mean of the data.
  2. The first PC v1\mathbf{v}_1v1​ is the direction that explains the most variance of the data.
  3. The second PC v2\mathbf{v}_2v2​ is the direction perpendicular to v1\mathbf{v}_1v1​ that explains the most variance.
  4. The third PC v3\mathbf{v}_3v3​ is the direction perpendicular to {v1,v2}\{\mathbf{v}_1, \mathbf{v}_2\}{v1​,v2​} that explains the most variance.
  5. …\ldots…

The formal algorithm can be described as follows,

  1. Data preprocessing: Compute μ=1M∑ix(i)\mathbf{\mu} = \frac{1}{M} \sum_i \mathbf{x}^{(i)}μ=M1​∑i​x(i) and replace each x(i)\mathbf{x}^{(i)}x(i) with x(i)−μ\mathbf{x}^{(i)} - \mathbf{\mu}x(i)−μ.
  2. Given pre-processed data matrix X∈RM×N\mathbf{X} \in \mathbb{R}^{M \times N}X∈RM×N, compute the sample covariance matrix Σ=1MXTX\mathbf{\Sigma} = \frac{1}{M} \mathbf{X}^T \mathbf{X}Σ=M1​XTX.
  3. Compute the KKK leading eigenvectors v1,…,vK\mathbf{v}_1, \ldots, \mathbf{v}_Kv1​,…,vK​ of Σ\mathbf{\Sigma}Σ where vi∈RN\mathbf{v}_i \in \mathbb{R}^Nvi​∈RN.
  4. Stack the eigenvectors together into an N×KN \times KN×K matrix V\mathbf{V}V where each column iii of V\mathbf{V}V corresponds to vi\mathbf{v}_ivi​.
  5. Project the matrix X\mathbf{X}X into the rank-KKK subspace of maximum variance by computing the matrix product Z=XV\mathbf{Z} = \mathbf{X} \mathbf{V}Z=XV.
  6. To reconstruct X\mathbf{X}X given Z\mathbf{Z}Z and V\mathbf{V}V, we use X^=ZVT\mathbf{\hat{X}} = \mathbf{Z} \mathbf{V}^TX^=ZVT.

How to Choose the Number of PCs?

We have two methods to set the number of components KKK.

  • Preserve some percentage of the variance, e.g., 95%.
  • Whatever works well for our final task (e.g., classification, regression).

Connection to SVD

We have seen that the minimum Frobenius norm linear dimensionality reduction problem could be solved using the rank-KKK SVD of X\mathbf{X}X,

arg⁡min⁡U,S,V∥X−USVT∥F2,\begin{align*} \underset{\mathbf{U}, \mathbf{S}, \mathbf{V}}{\arg \min} & \quad \Vert \mathbf{X} - \mathbf{U} \mathbf{S} \mathbf{V}^T \Vert_F^2, \end{align*}U,S,Vargmin​​∥X−USVT∥F2​,​

where U∈RM×K\mathbf{U} \in \mathbb{R}^{M \times K}U∈RM×K, S∈RK×K\mathbf{S} \in \mathbb{R}^{K \times K}S∈RK×K, and V∈RN×K\mathbf{V} \in \mathbb{R}^{N \times K}V∈RN×K.

The matrix Z=US\mathbf{Z} = \mathbf{U} \mathbf{S}Z=US gives the optimal rank-KKK representation of X\mathbf{X}X with respect to Frobenius norm minimization.

If we let K=NK = NK=N then X=USVT\mathbf{X} = \mathbf{U} \mathbf{S} \mathbf{V}^TX=USVT and XTX=VSTUTUSVT\mathbf{X}^T \mathbf{X} = \mathbf{V} \mathbf{S}^T \mathbf{U}^T \mathbf{U} \mathbf{S} \mathbf{V}^TXTX=VSTUTUSVT.

Due to orthogonality of U\mathbf{U}U, we get XTX=VS2VT\mathbf{X}^T \mathbf{X} = \mathbf{V} \mathbf{S}^2 \mathbf{V}^TXTX=VS2VT.

This means that the right singular vectors of X\mathbf{X}X are exactly the eigenvectors of XTX\mathbf{X}^T \mathbf{X}XTX.

We can also see that the eigenvalues of XTX\mathbf{X}^T \mathbf{X}XTX are the squares of the diagonal elements of S\mathbf{S}S.

This means that the KKK largest singular values and KKK largest eigenvalues correspond to the same KKK basis vectors.

According to PCA, the projection operation is Z=XV\mathbf{Z} = \mathbf{X} \mathbf{V}Z=XV, therefore,

Z=XV=(USVT)(V)=US.\mathbf{Z} = \mathbf{X} \mathbf{V} = (\mathbf{U} \mathbf{S} \mathbf{V}^T)(\mathbf{V}) = \mathbf{U} \mathbf{S}.Z=XV=(USVT)(V)=US.

Finally, note that if the decomposition are based only on the KKK leading principal vectors, the projections Z=XV\mathbf{Z} = \mathbf{X} \mathbf{V}Z=XV and Z=US\mathbf{Z} = \mathbf{U} \mathbf{S}Z=US will still be identical.

These manipulations show that PCA on XTX\mathbf{X}^T \mathbf{X}XTX and SVD on X\mathbf{X}X identify exactly the same subspace and result in exactly the same projection of the data into that subspace.

As a result, generic linear dimensionality reduction simultaneously minimizes the Frobenius norm of the reconstruction error of X\mathbf{X}X and maximizes the retained variance in the learned subspace.

Both SVD and PCA provide the same description of generic linear dimensionality reduction, an orthogonal basis for exactly the same optimal linear subspace.

When Does PCA Fail?

The primary motivation behind PCA is to decorrelate the dataset, i.e., remove second-order dependencies.

If higher-order dependencies exist between the features in the data, PCA may be insufficient at revealing all strucutre in the data.

PCA requires that each component must be perpendicular to the previous ones, but clearly, this requirement is overly stringent and the data might be arranged along non-orthogonal axes.

Kernel Principal Component Analysis

Kernel PCA (KPCA) is a non-linear extension of PCA that can be used to extract non-linear structure in the data.

Limitations of Linear Dimensionality Reduction

What if the data “lives” on a non-flat surface?

PCA can not capture the curvature of the data.

As usual, feature mapping can be used to overcome this.

Feature Mapping

If we apply a high-dimensional feature transformation to the data x(i)→ϕ(x(i))\mathbf{x}^{(i)} \rightarrow \phi(\mathbf{x}^{(i)})x(i)→ϕ(x(i)), we can project the high-dimensional data to a linear surface, i.e., run PCA on ϕ(X)\phi(\mathbf{X})ϕ(X). In the original space, the projection will be non-linear.

Feature Mapping + SVD

Given a data set X∈RM×N\mathbf{X} \in \mathbb{R}^{M \times N}X∈RM×N and a feature mapping function ϕ:RN↦RL\phi : \mathbb{R}^N \mapsto \mathbb{R}^Lϕ:RN↦RL for L>NL > NL>N, we obtain the following SVD-based algorithm,

  1. Compute U,S,V=SVD(ϕ(X))\mathbf{U}, \mathbf{S}, \mathbf{V} = \text{SVD}(\phi(\mathbf{X}))U,S,V=SVD(ϕ(X)).
  2. Return Z=US\mathbf{Z} = \mathbf{U} \mathbf{S}Z=US.

Feature Mapping + PCA

Given a data set X∈RM×N\mathbf{X} \in \mathbb{R}^{M \times N}X∈RM×N and a feature mapping function ϕ:RN↦RL\phi : \mathbb{R}^N \mapsto \mathbb{R}^Lϕ:RN↦RL for L>NL > NL>N, we obtain the following PCA-based algorithm,

  1. Compute Σ=1M∑i(ϕ(x(i)−μ))(ϕ(x(i)−μ))T\mathbf{\Sigma} = \frac{1}{M} \sum_i (\phi(\mathbf{x}^{(i)} - \mathbf{\mu})) (\phi(\mathbf{x}^{(i)} - \mathbf{\mu}))^TΣ=M1​∑i​(ϕ(x(i)−μ))(ϕ(x(i)−μ))T. where μ=1M∑iϕ(x(i))\mathbf{\mu} = \frac{1}{M} \sum_i \phi(\mathbf{x}^{(i)})μ=M1​∑i​ϕ(x(i)).

  2. Compute the KKK leading eigenvectors v1,…,vK\mathbf{v}_1, \ldots, \mathbf{v}_Kv1​,…,vK​ of Σ\mathbf{\Sigma}Σ where vj∈RL\mathbf{v}_j \in \mathbb{R}^Lvj​∈RL for j=1,…,Kj = 1, \ldots, Kj=1,…,K.

  3. Stack the eigenvectors together to form V=[v1,v2,…,vK]\mathbf{V} = [\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_K]V=[v1​,v2​,…,vK​], where V∈RL×K\mathbf{V} \in \mathbb{R}^{L \times K}V∈RL×K.

  4. Project the matrix ϕ(X)\phi(\mathbf{X})ϕ(X) into the rank-KKK subspace of maximum variance by computing the matrix product Z=ϕ(X)V\mathbf{Z} = \phi(\mathbf{X}) \mathbf{V}Z=ϕ(X)V.

Kernel PCA

As in classification, it becomes very expensive to use an explicit feature function to map data into a high-dimensional space.

In the basic SVD-based algortihm, there’s no way to avoid this problem.

However, in the PCA-based algorithm, we are able to take advantage of the kernel trick,

K(x(i),x(j))=ϕ(x(i))Tϕ(x(j)).\mathcal{K}(\mathbf{x}^{(i)}, \mathbf{x}^{(j)}) = \phi(\mathbf{x}^{(i)})^T \phi(\mathbf{x}^{(j)}).K(x(i),x(j))=ϕ(x(i))Tϕ(x(j)).

Given ϕ:RN↦RL\phi : \mathbb{R}^N \mapsto \mathbb{R}^Lϕ:RN↦RL, we can compute the covariance matrix in the new feature space,

Σ=1M∑j=1Mϕ(x(i))ϕ(x(j))T.\mathbf{\Sigma} = \frac{1}{M} \sum_{j = 1}^M \phi(\mathbf{x}^{(i)}) \phi(\mathbf{x}^{(j)})^T.Σ=M1​j=1∑M​ϕ(x(i))ϕ(x(j))T.

Note that we have assumed that the data is pre-centered in our new feature space. If this turns out to be false, we can center the data by subtracting the mean of the data in the new feature space.

Eigendecomposition of Σ\mathbf{\Sigma}Σ is given by,

Σvk=1M∑j=1Mϕ(x(i))ϕ(x(j))Tvk=λkvk,∀k=1,…,L.\mathbf{\Sigma} \mathbf{v_k} = \frac{1}{M} \sum_{j = 1}^M \phi(\mathbf{x}^{(i)}) \phi(\mathbf{x}^{(j)})^T \mathbf{v}_k = \lambda_k \mathbf{v}_k, \forall k = 1, \ldots, L.Σvk​=M1​j=1∑M​ϕ(x(i))ϕ(x(j))Tvk​=λk​vk​,∀k=1,…,L.

It is not hard to see that vk\mathbf{v}_kvk​ can be expressed as,

vk=∑j=1Mwk(j)ϕ(x(j)),\mathbf{v_k} = \sum_{j = 1}^M w_k^{(j)} \phi(\mathbf{x}^{(j)}),vk​=j=1∑M​wk(j)​ϕ(x(j)),

where wk(j)=1Mλkϕ(x(j))Tvkw_k^{(j)} = \frac{1}{M \lambda_k} \phi(\mathbf{x}^{(j)})^T \mathbf{v}_kwk(j)​=Mλk​1​ϕ(x(j))Tvk​.

So, the kernel PCA is a linear combination of high-dimensional vectors, and wk(j)w_k^{(j)}wk(j)​ are weights to be determined.

If we left-multiply ϕ(x(i))T\phi(\mathbf{x}^{(i)})^Tϕ(x(i))T to both sides, we have,

ϕ(x(i))Tvk=∑j=1Mwk(j)ϕ(x(i))Tϕ(x(j))=Mλkwk(i).\phi(\mathbf{x}^{(i)})^T \mathbf{v_k} = \sum_{j = 1}^M w_k^{(j)} \phi(\mathbf{x}^{(i)})^T \phi(\mathbf{x}^{(j)}) = M \lambda_k w_k^{(i)}.ϕ(x(i))Tvk​=j=1∑M​wk(j)​ϕ(x(i))Tϕ(x(j))=Mλk​wk(i)​.

Defining the kernel matrix K∈RM×M\mathbf{K} \in \mathbb{R}^{M \times M}K∈RM×M, where Kij=K(x(i),x(j))=ϕ(x(i))Tϕ(x(j))K_{ij} = \mathcal{K}(\mathbf{x}^{(i)}, \mathbf{x}^{(j)}) = \phi(\mathbf{x}^{(i)})^T \phi(\mathbf{x}^{(j)})Kij​=K(x(i),x(j))=ϕ(x(i))Tϕ(x(j)).

Then,

∑j=1MKijwk(j)=Mλkwk(i).\sum_{j = 1}^M K_{ij} w_k^{(j)} = M \lambda_k w_k^{(i)}.j=1∑M​Kij​wk(j)​=Mλk​wk(i)​.

If we consider i=1,…,Mi = 1, \ldots, Mi=1,…,M, the above scalar equation becomes the iii-th component of the following vector equation,

Kwk=Mλkwk,\mathbf{K} \mathbf{w}_k = M \lambda_k \mathbf{w}_k,Kwk​=Mλk​wk​,

where wk=[wk(1),wk(2),…,wk(M)]T\mathbf{w}_k = [w_k^{(1)}, w_k^{(2)}, \ldots, w_k^{(M)}]^Twk​=[wk(1)​,wk(2)​,…,wk(M)​]T is the kkk-th eigenvector of K\mathbf{K}K.

MλkM \lambda_kMλk​ is the eigenvalue of K\mathbf{K}K, which is proportional to the eigenvalue λk\lambda_kλk​ of the covariance matrix Σ\mathbf{\Sigma}Σ in the feature space.

Therefore, PCA on Σ\mathbf{\Sigma}Σ is equivalent to PCA on K\mathbf{K}K.

For a new point x⋆\mathbf{x}^{\star}x⋆, the kkk-th kernel PC can be obtained by projection ϕ(x⋆)\phi(\mathbf{x}^{\star})ϕ(x⋆) on the kkk-th eigenvector vk\mathbf{v}_kvk​ of Σ\mathbf{\Sigma}Σ.

ϕ(x⋆)Tvk=∑i=1wk(i)ϕ(x⋆)Tϕ(x(i))=∑i=1wk(i)K(x⋆,x(i)).\phi(\mathbf{x}^{\star})^T \mathbf{v_k} = \sum_{i = 1} w_k^{(i)} \phi(\mathbf{x}^{\star})^T \phi(\mathbf{x}^{(i)}) = \sum_{i = 1} w_k^{(i)} \mathcal{K}(\mathbf{x}^{\star}, \mathbf{x}^{(i)}).ϕ(x⋆)Tvk​=i=1∑​wk(i)​ϕ(x⋆)Tϕ(x(i))=i=1∑​wk(i)​K(x⋆,x(i)).

Kernel PCA Algorithm

Given a data set X∈RM×N\mathbf{X} \in \mathbb{R}^{M \times N}X∈RM×N and a kernel function K\mathcal{K}K, kernel PCA can be computed as follows,

  1. Compute Kij=K(x(i),x(j))K_{ij} = \mathcal{K}(\mathbf{x}^{(i)}, \mathbf{x}^{(j)})Kij​=K(x(i),x(j)) for all i,ji,ji,j.

  2. Compute K′=(I−1M)K(I−1M)\mathbf{K}^{\prime} = (\mathbf{I} - \mathbf{1}_M) \mathbf{K} (\mathbf{I} - \mathbf{1}_M)K′=(I−1M​)K(I−1M​) where 1M\mathbf{1}_M1M​ is an M×MM \times MM×M matrix where every entry is 1M\frac{1}{M}M1​. The goal is to zero center data points in the feature space.

  3. Compute the KKK leading eigenvectors w1,…,wK\mathbf{w}_1, \ldots, \mathbf{w}_Kw1​,…,wK​ of K′\mathbf{K}^{\prime}K′ along with their eigenvalues Mλ1,…,MλKM \lambda_1, \ldots, M \lambda_KMλ1​,…,MλK​.

  4. Compute the kkk-th PC of the projected data vector z∈RK×1\mathbf{z} \in \mathbb{R}^{K \times 1}z∈RK×1,

zk=∑i=1Mwk(i)K(x,x(i)).z_k = \sum_{i = 1}^M w_k^{(i)} \mathcal{K}(\mathbf{x}, \mathbf{x}^{(i)}).zk​=i=1∑M​wk(i)​K(x,x(i)).

Summary

Kernel PCA uses the kernel trick to peform PCA in high-dimensional space.

  • Coeffcients are based on a non-linear projection of the data.
  • The type of projection is based on the kernel function selected.

Using RBF kernel, KPCA can split the data into clusters.

Kernel PCA can provide an effective pre-processing step for clustering methods as well as linear classification and regression methods.

However, exact compuation of kernel PCA can be expensive because the size of the matrix to be decomposed is M×MM \times MM×M.
