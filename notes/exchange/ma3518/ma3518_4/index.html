Introduction

The main objective of many statistical investigations is to make predictions.

For instance, an engineer may wish to predict the amount of oxide that will form on the surface of a metal baked in an oven for one hour at 200∘^{\circ}∘C, or the amount of deformation of a ring subjected to a certain compressive force.

Usually, such predictions require a formula which relates the dependent variable whose value we want to predict (or called response) to one or more other variables, usually called predictors (or regressors).

The collection of statistical tools for this is called regression analysis.

Simple Linear Regression

The simplest form of regression analysis is simple linear regression, which involves a single predictor and a single response variable.

Y=β0+β1X+eY = \beta_0 + \beta_1 X + eY=β0​+β1​X+e

where:

  • YYY is the response variable.
  • XXX is the predictor variable.
  • β0\beta_0β0​ is the intercept.
  • β1\beta_1β1​ is the slope.

Assume that E(e)=0\mathbb{E}(e) = 0E(e)=0 and Var(e)=σ2\text{Var}(e) = \sigma^2Var(e)=σ2.

Suppose we fix X=xX = xX=x, meaning

Y=β0+β1x+e,Y = \beta_0 + \beta_1 x + e,Y=β0​+β1​x+e,

with mean E(Y)=β0+β1x\mathbb{E}(Y) = \beta_0 + \beta_1 xE(Y)=β0​+β1​x and variance Var(Y)=σ2\text{Var}(Y) = \sigma^2Var(Y)=σ2 (YYY is normal if eee is normal).

The linear function β0+β1x\beta_0 + \beta_1 xβ0​+β1​x is thus the function giving the mean value of YYY for each possible value of XXX.

It is called the regression function (or regression line) and will be denoted as,

μY∣X=x=E(Y∣X=x)=β0+β1x.\mu_{Y | X = x} = \mathbb{E}(Y | X = x) = \beta_0 + \beta_1 x.μY∣X=x​=E(Y∣X=x)=β0​+β1​x.

The slope β1\beta_1β1​ is the change in mean of YYY for one unit change in XXX, the intercept β0\beta_0β0​ is the mean value of YYY when X=0X = 0X=0.

Least Squares Estimator

Gauss proposed estimating β0\beta_0β0​ and β1\beta_1β1​ to minimize the sum of the squares of the vertical deviations between the observed responses and the fitted line.

These deviations are called the residuals.

Analysis of Variance (ANOVA)

The most interesting test for simple linear regression is probably,

H0:β1=0vs.H1:β1≠0.H_0 : \beta_1 = 0 \quad \text{vs.} \quad H_1 : \beta_1 \neq 0.H0​:β1​=0vs.H1​:β1​=0.

β1^s/sxx∼tn−2.\frac{\hat{\beta_1}}{s / \sqrt{s_{xx}}} \sim t_{n - 2}.s/sxx​​β1​^​​∼tn−2​.

Firstly, let’s define some quantities,

SST=∑i(yi−yˉ)2SSreg=∑i(y^i−yˉ)2RSS=∑i(yi−y^i)2=∑ie^i2\begin{align*} SST = \sum_i (y_i - \bar{y})^2 \newline SS_{\text{reg}} = \sum_i (\hat{y}_i - \bar{y})^2 \newline RSS = \sum_i (y_i - \hat{y}_i)^2 = \sum_i \hat{e}_i^2 \end{align*}SST=i∑​(yi​−yˉ​)2SSreg​=i∑​(y^​i​−yˉ​)2RSS=i∑​(yi​−y^​i​)2=i∑​e^i2​​

Coefficient of Determination

Suppose SST≃SSregSST \simeq SS_{\text{reg}}SST≃SSreg​ and RSS≃0RSS \simeq 0RSS≃0.

Almost the whole variation is due to error, and the regression is not significant.

The coefficient of determination is defined as,

R2=SSregSST,R^2 = \frac{SS_{\text{reg}}}{SST},R2=SSTSSreg​​,

which represents the proportion of the variability in the responses that is explained by the predictor.

A value of R2R^2R2 near 1 indicates a good fit to the data.

A value of R2R^2R2 near 0 indicates a poor fit to the data.

Example

A food processing center needs to be able to switch from one type of package to another quickly to react to changes in order patterns. Consultants have developed a new method for changing the production line. We have a sample of 48 change-over times using the new method, and an independent sample of 72 change-over times for the existing method.

We model the relationship between YYY, the change-over time and XXX, the dummy variable for the new method.

We consider the simple linear regression model,

Y=β0+β1X+e,Y = \beta_0 + \beta_1 X + e,Y=β0​+β1​X+e,

where Y=Y = Y= change over time and xxx is the dummy variable (i.e., x = 1 for the new change-over method and 0 for the existing method).

Leverage Points

A leverage point is a point whose xxx-value is distant from the other xxx-values.

A point is a bad leverage point if its yyy-value does not follow the pattern set by the other data points. In other words, a bad leverage point is a leverage point which is also an outlier.

The mathematical definition of leverage is,

yi^=∑j=1nhijyj,\hat{y_i} = \sum_{j = 1}^n h_{ij} y_j,yi​^​=j=1∑n​hij​yj​,

where hij=[1n+(xi−xˉ)(xj−xˉ)Sxx]h_{ij} = \left[\frac{1}{n} + \frac{(x_i - \bar{x})(x_j - \bar{x})}{S_{xx}}\right]hij​=[n1​+Sxx​(xi​−xˉ)(xj​−xˉ)​].

We can see that, if we sum up hijh_{ij}hij​ for all jjj,

∑j=1nhij=1.\sum_{j = 1}^n h_{ij} = 1.j=1∑n​hij​=1.

Thus,

hii=1n+(xi−xˉ)2∑j=1n(xj−xˉ)2,h_{ii} = \frac{1}{n} + \frac{(x_i - \bar{x})^2}{\sum_{j = 1}^n (x_j - \bar{x})^2},hii​=n1​+∑j=1n​(xj​−xˉ)2(xi​−xˉ)2​,

is called the leverage of the iii-th data point. If hii≈1,yi^≈yih_{ii} \approx 1, \hat{y_i} \approx y_ihii​≈1,yi​^​≈yi​.

A popular rule of thumb for detecting leverage points is,

hii>2×average(hii).=2×2n=4n.h_{ii} > 2 \times \text{average}(h_{ii}). = 2 \times \frac{2}{n} = \frac{4}{n}.hii​>2×average(hii​).=2×n2​=n4​.

When dealing with bad leverage points, after finding them we will remove them and then fit a different model.

Non-constant Variance

What do we do if the variance is not constant?

We can use a transformation, the transformation is used to overcome problems due to non-constant variance but also due to nonlinearity.

The methods that we will discuss deal with positive variables only.

There is no general rule of transformation, if for example we have count data, this is often modelled by the Poisson distribution. Thus, the appropriate transformation is square root.
