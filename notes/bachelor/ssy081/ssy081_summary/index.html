Signals

A signal is a set of information or data. Any physical quantity that varies over time, space or any other variable or variables.

Even, Odd & Periodic

An even function is symmetrical about the vertical axis. Mathematically this means:

f(t)=f(−t)f[k]=f[−k]f(t) = f(-t) \newline f[k] = f[-k]f(t)=f(−t)f[k]=f[−k]

An odd function is anti-symmetrical about the vertical axis. Mathematically this means:

f(t)=−f(−t)f[k]=−f[−k]f(t) = -f(-t) \newline f[k] = -f[-k]f(t)=−f(−t)f[k]=−f[−k]

A periodic function has a fundamental period (minimum), T0T_0T0​. Which also means it has a fundamental frequency, f0=1T0f_0 = \dfrac{1}{T_0}f0​=T0​1​.

f(t)=f(t+nT0)f[k]=f[k+nK0]f(t) = f(t + nT_0) \newline f[k] = f[k + nK_0]f(t)=f(t+nT0​)f[k]=f[k+nK0​]

We sometimes define the fundamental frequency in angular velocity instead of Hz, which means, ω0=2πf0=2πT0\omega_0 = 2\pi f_0 = \dfrac{2\pi}{T_0}ω0​=2πf0​=T0​2π​.

Transfer function

Say we have a system which can be modeled as:

y(t)=f(t)∗h(t)y(t) = f(t) * h(t)y(t)=f(t)∗h(t)

The same system, but in the frequency domain is therefore:

Y(ω)=F(ω) H(ω)Y(\omega) = F(\omega)\ H(\omega)Y(ω)=F(ω) H(ω)

Just as we call, h(t)h(t)h(t), the system response function, we call H(ω)H(\omega)H(ω) for the spectral response of the system.

Since, in the most general case, we view our frequency functions as complex functions. We can say that

An input signals spectral component of frequency, ω\omegaω, is modified in amplitude by a factor of, ∣H(ω)∣|H(\omega)|∣H(ω)∣ and shifted in phase by an angle of ∠H(ω)\angle H(\omega)∠H(ω)

However, since we modify the phase, this means that our output signal may have a different waveform than our input.
