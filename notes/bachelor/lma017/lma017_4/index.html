Introduction

In this part we’ll define the chain rule for functions with several variables. We will also cover something called implicit differentiation.

Chain rule

Let’s recall the chain rule for functions with one variable.

f(x)=h(g(x))f′(x)=h′(g(x))⋅g′(x)f(x) = h(g(x)) \newline f'(x) = h'(g(x)) \cdot g'(x)f(x)=h(g(x))f′(x)=h′(g(x))⋅g′(x)

Alternative notation:

y=f(x)x=g(t)y = f(x) \newline x = g(t)y=f(x)x=g(t) dydt=dydx⋅dxdt\dfrac{dy}{dt} = \dfrac{dy}{dx} \cdot \dfrac{dx}{dt}dtdy​=dxdy​⋅dtdx​

Chain Rule for functions with two variables

dzdt=∂z∂xdxdt+∂z∂ydydt\dfrac{dz}{dt} = \dfrac{\partial z}{\partial x} \dfrac{dx}{dt} + \dfrac{\partial z}{\partial y} \dfrac{dy}{dt}dtdz​=∂x∂z​dtdx​+∂y∂z​dtdy​ dzds=∂z∂xdxds+∂z∂ydyds\dfrac{dz}{ds} = \dfrac{\partial z}{\partial x} \dfrac{dx}{ds} + \dfrac{\partial z}{\partial y} \dfrac{dy}{ds}dsdz​=∂x∂z​dsdx​+∂y∂z​dsdy​

Implicit differentiation

Sometimes the function is not given explicitly, rather implicitly. For example:

Let y(x) be a function that satisfies:

x3+3xy−y5+1=0x^3 + 3xy - y^5 + 1 = 0x3+3xy−y5+1=0

How can we find dydx\dfrac{dy}{dx}dxdy​?

The first logical answer would be to explicitly find y(x)y(x)y(x), but that can be really tedious. We can instead view this equation as a function of two variables.

Let’s introduce a new function, F(x,y)F(x, y)F(x,y). This means:

F(x,y)=x3+3xy−y5+1=0F(x, y) = x^3 + 3xy - y^5 + 1 = 0F(x,y)=x3+3xy−y5+1=0

We want to find a y(x)y(x)y(x) such that this equation is satisfied. Since y(x)y(x)y(x) depends on xxx, and xxx in itself depends on xxx. We can apply the chain rule!

dF(x,y)dx=∂F∂xdxdx+∂F∂ydydx\dfrac{dF(x, y)}{dx} = \dfrac{\partial F}{\partial x} \dfrac{dx}{dx} + \dfrac{\partial F}{\partial y} \dfrac{dy}{dx}dxdF(x,y)​=∂x∂F​dxdx​+∂y∂F​dxdy​

We want to find y(x)y(x)y(x) such that:

∂F∂xdxdx+∂F∂ydydx=0\dfrac{\partial F}{\partial x} \dfrac{dx}{dx} + \dfrac{\partial F}{\partial y} \dfrac{dy}{dx} = 0∂x∂F​dxdx​+∂y∂F​dxdy​=0

Which means:

dydx=−∂F∂x∂F∂y=−FxFy\dfrac{dy}{dx} = - \dfrac{\dfrac{\partial F}{\partial x}}{\dfrac{\partial F}{\partial y}} = \boxed{-\dfrac{F_x}{F_y}}dxdy​=−∂y∂F​∂x∂F​​=−Fy​Fx​​​

Here we need to assume FyF_yFy​ isn’t zero. But this means in our case that:

dydx=−3x2+3y3x−5y4\dfrac{dy}{dx} = - \dfrac{3x^2 + 3y}{3x - 5y^4}dxdy​=−3x−5y43x2+3y​

dydx=−FxFy\boxed{\dfrac{dy}{dx} = -\dfrac{F_x}{F_y}}dxdy​=−Fy​Fx​​​
