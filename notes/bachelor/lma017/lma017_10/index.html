Introduction

In this part we’ll cover how to compute the double integral over general regions.

Double integrals over general regions

Let f(x,y)f(x, y)f(x,y) be defined on the domain, DDD, where DDD is a bounded region. DDD is a general region, meaning it can have any geometrical shape.

However, if we reduce our problem into something we know, a rectangular region, we know how to compute the double integral.

Therefore, let’s enclose DDD, with a rectangle, let’s call this rectangle for RRR.

We define a function, f~\tilde{f}f~​, on RRR as the following:

f~(x,y)={f(x,y)(x,y)∈D0(x,y)∉D\tilde{f}(x, y) = \begin{cases} f(x, y) & (x, y) \in D \newline 0 & (x, y) \notin D \end{cases}f~​(x,y)={f(x,y)0​(x,y)∈D(x,y)∈/D​

Definition

The double integral of, fff over DDD is:

∬Df(x,y) dA=∬Rf~(x,y) dA\iint_D f(x, y)\ dA = \iint_R \tilde{f}(x, y)\ dA∬D​f(x,y) dA=∬R​f~​(x,y) dA

Equivalent definition

Recall our original definition of the double integral, we can also divide the domain, DDD, into smaller pieces.

Therefore, the double integral of, fff, over DDD is also:

∬Df(x,y) dA=lim⁡Δx→0Δy→0∑j=1m∑i=1nf(xi\*,yj\*)(ΔA)ij\iint_{D} f(x, y)\ dA = \lim_{\substack{\Delta x \to 0 \newline \Delta y \to 0}} \sum_{j = 1}^{m} \sum_{i = 1}^{n} f(x_{i}^{\*}, y_{j}^{\*}) (\Delta A)_{ij}∬D​f(x,y) dA=Δx→0Δy→0​lim​j=1∑m​i=1∑n​f(xi\*​,yj\*​)(ΔA)ij​

Since everything outside DDD, is 0, they contribute nothing to the overall volume. Therefore, the volume under f~\tilde{f}f~​ is the same as the volume under fff.

How to compute general double integrals

To compute double integrals over a general region, we firstly need to introduce a new concept.

Type I and II regions

A domain, DDD, is of type I, if it lies between two graphs of two continuous functions of x.

Meaning:

D={(x,y)∣a≤x≤b,g1(x)≤y≤g2(x)}D = \{(x, y) | a \leq x \leq b, g_1(x) \leq y \leq g_2(x)\}D={(x,y)∣a≤x≤b,g1​(x)≤y≤g2​(x)}

Let’s see how we can compute this now. Let DDD be of type I. Let’s enclose this region within a rectangle as well:

∬f(x,y) dA=∬Rf~(x,y) dA=∫ab∫cdf~(x,y) dy dx\iint f(x, y)\ dA = \iint_R \tilde{f}(x, y)\ dA = \int_a^b \int_c^d \tilde{f}(x, y)\ dy\ dx∬f(x,y) dA=∬R​f~​(x,y) dA=∫ab​∫cd​f~​(x,y) dy dx

If we fix xxx, then f~\tilde{f}f~​ becomes:

f~(x,y)={0c≤y≤g1(x)f(x,y)g1(x)≤y≤g2(x)0g2(x)≤y≤d\tilde{f}(x, y) = \begin{cases} 0 & c \leq y \leq g_1(x) \newline f(x, y) & g_1(x) \leq y \leq g_2(x) \newline 0 & g_2(x) \leq y \leq d \end{cases}f~​(x,y)=⎩⎨⎧​0f(x,y)0​c≤y≤g1​(x)g1​(x)≤y≤g2​(x)g2​(x)≤y≤d​

Which means:

∫ab∫g1(x)g2(x)f(x,y) dy dx\boxed{\int_a^b \int_{g_1(x)}^{g_2(x)} f(x, y)\ dy\ dx}∫ab​∫g1​(x)g2​(x)​f(x,y) dy dx​

Type I definition

If DDD is of type I, then:

∬Df(x,y) dA=∫ab∫g1(x)g2(x)f(x,y) dy dx\iint_D f(x, y)\ dA = \int_a^b \int_{g_1(x)}^{g_2(x)} f(x, y)\ dy\ dx∬D​f(x,y) dA=∫ab​∫g1​(x)g2​(x)​f(x,y) dy dx

Type II

D is of type II, if it lies between two graphs of two continuous functions of y.

D={(x,y)∣h1(y)≤x≤h2(y),c≤y≤d}D = \{(x, y) | h_1(y) \leq x \leq h_2(y), c \leq y \leq d\}D={(x,y)∣h1​(y)≤x≤h2​(y),c≤y≤d}

We perform the same procedure as type I, instead of fixing xxx, we fix yyy.

Type II definition

If DDD is of type II, then:

∬Df(x,y) dA=∫cd∫h1(y)h2(y)f(x,y) dx dy\iint_D f(x, y)\ dA = \int_c^d \int_{h_1(y)}^{h_2(y)} f(x, y)\ dx\ dy∬D​f(x,y) dA=∫cd​∫h1​(y)h2​(y)​f(x,y) dx dy

Before we head into doing a few examples, it’s a good practice always draw where our DDD lies. So we easily get a geometrical sense of how it lies.

Changing order of integral

One might now think that we can change the order of integration, using Fubini’s Theorem.

That is not the case.

Say we have:

∫g1(x)g2(x)∫abf(x,y) dx dy\int_{g_1(x)}^{g_2(x)} \int_a^b f(x, y)\ dx\ dy∫g1​(x)g2​(x)​∫ab​f(x,y) dx dy

Computing this entire integral will not yield a singular number, it will yield a function of xxx instead.

However, if DDD is both type I and II, we can change order of integration, but we need to change our limits accordingly.

Example

Find ∬Dsin(y2) dA\iint_D sin(y^2)\ dA∬D​sin(y2) dA, where DDD is the region bounded by, x=0,y=1,y=xx = 0, y = 1, y = xx=0,y=1,y=x
