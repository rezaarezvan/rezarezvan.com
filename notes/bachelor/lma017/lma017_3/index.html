Introduction

In this part we’ll define higher partial derivatives and how we define if a function with several variables is differentiable.

Higher partial derivatives

In the last part we defined what partial derivatives were:

fx=lim⁡h→0f(x+h,y)−f(x,y)hf_x = \lim{h \to 0} \dfrac{f(x + h, y) - f(x, y)}{h}fx​=limh→0hf(x+h,y)−f(x,y)​

We can now derive this function again:

fxx=lim⁡h→0fx(x+h,y)−fx(x,y)hf_{xx} = \lim{h \to 0} \dfrac{f_x(x + h, y) - f_x(x, y)}{h}fxx​=limh→0hfx​(x+h,y)−fx​(x,y)​ fxy=lim⁡h→0fx(x,y+h)−fx(x,y)hf_{xy} = \lim{h \to 0} \dfrac{f_x(x, y + h) - f_x(x, y)}{h}fxy​=limh→0hfx​(x,y+h)−fx​(x,y)​ fyx=lim⁡h→0fy(x+h,y)−fy(x,y)hf_{yx} = \lim{h \to 0} \dfrac{f_y(x + h, y) - f_y(x, y)}{h}fyx​=limh→0hfy​(x+h,y)−fy​(x,y)​ fyy=lim⁡h→0fy(x,y+h)−fy(x,y)hf_{yy} = \lim{h \to 0} \dfrac{f_y(x, y + h) - f_y(x, y)}{h}fyy​=limh→0hfy​(x,y+h)−fy​(x,y)​

Example

Find all 2nd order partial derivatives of:

f(x,y)=x⋅eyf(x, y) = x \cdot e^yf(x,y)=x⋅ey

1st order:

fx(x,y)=eyf_x(x, y) = e^yfx​(x,y)=ey fy(x,y)=x⋅eyf_y(x, y) = x \cdot e^yfy​(x,y)=x⋅ey

2nd order:

fxx(x,y)=0f_{xx}(x, y) = 0fxx​(x,y)=0 fxy(x,y)=eyf_{xy}(x, y) = e^yfxy​(x,y)=ey fyx(x,y)=eyf_{yx}(x, y) = e^yfyx​(x,y)=ey fyy(x,y)=x⋅eyf_{yy}(x, y) = x \cdot e^yfyy​(x,y)=x⋅ey

As we see fxyf_{xy}fxy​ and fyxf_{yx}fyx​ are the same. This is often the case.

Clairaut’s theorem

Clairaut’s theorem states that:

If fxyf_{xy}fxy​ and fyxf_{yx}fyx​ are continuous, then fxy=fyxf_{xy} = f_{yx}fxy​=fyx​

We’ll save the actual proof for another day, but we can “explain” why this theorem holds:

Explaination

Suppose we have:

f(x,y)=xnymf(x, y) = x^n y^mf(x,y)=xnym

The 1st & 2nd order partial derivatives of this polynomial are:

fx(x,y)=n⋅xn−1ymf_x(x, y) = n \cdot x^{n - 1} y^mfx​(x,y)=n⋅xn−1ym fy(x,y)=m⋅xnym−1f_y(x, y) = m \cdot x^n y^{m - 1}fy​(x,y)=m⋅xnym−1 fxy(x,y)=m⋅n⋅xn−1ym−1f_{xy}(x, y) = m \cdot n \cdot x^{n - 1} y^{m - 1}fxy​(x,y)=m⋅n⋅xn−1ym−1 fyx(x,y)=n⋅m⋅xn−1ym−1f_{yx}(x, y) = n \cdot m \cdot x^{n - 1} y^{m - 1}fyx​(x,y)=n⋅m⋅xn−1ym−1

We see that this holds true for polynomials. Since any function can be approximated by polynomials, it’s not surprising Clairaut’s theorem holds.

Remark

Suppose, fff, is a composition of sin(x)sin(x)sin(x), cos(x)cos(x)cos(x), polynomials and the exponential function. For example:

f(x,y)=ecos(xy−3⋅sin(xy))f(x, y) = e^{cos(xy - 3 \cdot sin(xy))}f(x,y)=ecos(xy−3⋅sin(xy))

Since all of these functions are differentiable any amount of times and all of these are continuous. This must mean that partial derivatives of any order must also be continuous.

This means we can easily use Clairaut’s theorem, since it only applies to functions that are continuous. Therefore we do not need to know if the partial derivatives are continuous ahead of time.

Example

Find fxyf_{xy}fxy​:

f(x,y)=sin(ex⋅cos(x−3))+xyf(x, y) = sin(e^{x \cdot cos(x - 3))} + xyf(x,y)=sin(ex⋅cos(x−3))+xy

Trying to first compute fxf_xfx​ will prove to be quite tricky, but using Clairaut’s theorem this problem is trivial:

fy(x,y)=xf_y(x, y) = xfy​(x,y)=x fyx(x,y)=1\boxed{f_yx(x, y) = 1}fy​x(x,y)=1​

Notation

There are a lot of different notations for partial derivatives. We’ll mainly use fxf_xfx​ and fyf_yfy​.

However, the most common notation is:

fx=∂f∂xf_x = \dfrac{\partial f}{\partial x}fx​=∂x∂f​ fy=∂f∂yf_y = \dfrac{\partial f}{\partial y}fy​=∂y∂f​

In the case for higher order partial derivatives:

fxx=∂(∂f∂x )∂x=∂2f∂x2f_{xx} = \dfrac{\partial \left(\dfrac{\partial f}{\partial x}\ \right)}{\partial x} = \dfrac{\partial^2 f}{\partial x^2}fxx​=∂x∂(∂x∂f​ )​=∂x2∂2f​ fxy=∂(∂f∂x )∂y=∂2f∂x∂yf_{xy} = \dfrac{\partial \left(\dfrac{\partial f}{\partial x}\ \right)}{\partial y} = \dfrac{\partial^2 f}{\partial x \partial y}fxy​=∂y∂(∂x∂f​ )​=∂x∂y∂2f​ fyx=∂(∂f∂y )∂x=∂2f∂y∂xf_{yx} = \dfrac{\partial \left(\dfrac{\partial f}{\partial y}\ \right)}{\partial x} = \dfrac{\partial^2 f}{\partial y \partial x}fyx​=∂x∂(∂y∂f​ )​=∂y∂x∂2f​ fyy=∂(∂f∂y )∂y=∂2f∂y2f_{yy} = \dfrac{\partial \left(\dfrac{\partial f}{\partial y}\ \right)}{\partial y} = \dfrac{\partial^2 f}{\partial y^2}fyy​=∂y∂(∂y∂f​ )​=∂y2∂2f​

N’th order partial derivatives

We can derive how many times we want, with how many variables we want:

fxxx,fxxy,fxyx,…f_{xxx}, f_{xxy}, f_{xyx}, \ldotsfxxx​,fxxy​,fxyx​,…

This also includes Clairaut’s theorem.

Geometrical sense

Let’s try to understand what derivatives in two variables looks like geometricaly.

For one variable, it is: f′(a)=f'(a) =f′(a)= the slope of the tangent line to the graph at (a,f(a))(a, f(a))(a,f(a)). We could also say that this is the rate of change at point aaa.

For two variables: fx(a,b)f_x(a, b)fx​(a,b) intersects the graph by the vertical plane, y=by = by=b. CCC, is the curve of the intersection. In other words fx(a,b)f_x(a, b)fx​(a,b) = the slope of the tangent line TTT.

Differentiability

For one function, the following statement holds:

fff is differentiable ⇒\Rightarrow⇒ fff has derivate at aaa, meaning f′(a)f'(a)f′(a) exists.

When dealing with several variables, this becomes tricky. Let’s look at the definition and see what we can do:

f′(a)=lim⁡x→0f(x+h)−f(x)hf'(a) = \lim_{x \to 0} \dfrac{f(x + h) - f(x)}{h}f′(a)=x→0lim​hf(x+h)−f(x)​

This just means we have a small change in xxx, so let’s say:

Δx=h\Delta x = hΔx=h

Let’s define the small change in the function value as:

Δy:=f(a+Δx)−f(a)\Delta y := f(a + \Delta x) - f(a)Δy:=f(a+Δx)−f(a)

Then, the derivate is:

f′(a)=lim⁡Δx→0ΔyΔxf'(a) = \lim_{\Delta x \to 0} \dfrac{\Delta y}{\Delta x}f′(a)=Δx→0lim​ΔxΔy​

Let’s move around the terms:

lim⁡Δx→0ΔyΔx−f′(a)=0\lim_{\Delta x \to 0} \dfrac{\Delta y}{\Delta x} - f'(a) = 0Δx→0lim​ΔxΔy​−f′(a)=0

Let’s now call this for ε\varepsilonε:

ε=ΔyΔx−f′(a)\varepsilon = \dfrac{\Delta y}{\Delta x} - f'(a)ε=ΔxΔy​−f′(a)

We can define this as a function of Δx\Delta xΔx:

ε=ε(Δx)\varepsilon = \varepsilon(\Delta x)ε=ε(Δx)

Which means:

εΔx=Δy−f′(a)Δx\varepsilon \Delta x = \Delta y - f'(a) \Delta xεΔx=Δy−f′(a)Δx

Which finally means:

Δy=εΔx+f′(a)Δx\boxed{\Delta y = \varepsilon \Delta x + f'(a) \Delta x}Δy=εΔx+f′(a)Δx​

And as Δx→0\Delta x \to 0Δx→0, ε→0\varepsilon \to 0ε→0 as well.

For functions of two variables, we can now instead define Δz\Delta zΔz:

Δz=…=fx(a,b)Δx+fy(a,b)Δy+ε1Δx+ε2Δy\Delta z = \ldots = \boxed{f_x(a, b)\Delta x + f_y(a, b)\Delta y + \varepsilon_1 \Delta x + \varepsilon_2 \Delta y}Δz=…=fx​(a,b)Δx+fy​(a,b)Δy+ε1​Δx+ε2​Δy​

Where:

ε1=ε1(Δx,Δy),ε2=ε2(Δx,Δy)\varepsilon_1 = \varepsilon_1(\Delta x, \Delta y), \varepsilon_2 = \varepsilon_2(\Delta x, \Delta y)ε1​=ε1​(Δx,Δy),ε2​=ε2​(Δx,Δy)
