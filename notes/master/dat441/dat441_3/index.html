Successive Elimination (K>2K > 2K>2)

In the last part we introduced successive elimination for the case of K=2K=2K=2 arms.

We will extend this to the case of K>2K > 2K>2 arms and introduce another adaptive exploration algorithm.

[!SUMMARY/Successive Elimination Algorithm (K>2K > 2K>2)] Initially all arms are set “active”;

Each phase,

  • Try all active arms (thus each phase may contain multiple rounds);
  • Deactivate all arms aaa such that ∃a′\exists a^{\prime}∃a′ with UCBt(a)<LCBt(a′)\mathrm{UCB}_t(a) < \mathrm{LCB}_t(a^{\prime})UCBt​(a)<LCBt​(a′);

Repeat until only one arm remains active.

Further, let a⋆a^{\star}a⋆ be an optimal arm, and consider any arm aaa such that μ(a)<μ(a⋆)\mu(a) < \mu(a^{\star})μ(a)<μ(a⋆).

If we analyze the last round t^\hat{t}t^ when we have not yet deactivated arm aaa (or, the last round TTT if aaa is still active at the end).

The confidence intervals of the two arms aaa and a⋆a^{\star}a⋆ at round t≤t^t \leq \hat{t}t≤t^ must overlap (otherwise, arm aaa would have been deactivated),

Δ(a):−μ(a⋆)−μ(a)≤2rt(a)+2rt(a⋆)⏟≤O(rt(a))\Delta(a) \coloneq \mu(a^{\star}) - \mu(a) \leq \underbrace{2 r_t(a) + 2 r_t(a^{\star})}_{\leq \mathcal{O}(r_t(a))}Δ(a):−μ(a⋆)−μ(a)≤≤O(rt​(a))2rt​(a)+2rt​(a⋆)​​

As we established, arm aaa is never played after round t^\hat{t}t^, then,

nt^(a)=nT(a)rt^(a)=rT(a)\begin{align*} n_{\hat{t}}(a) & = n_T(a) \newline r_{\hat{t}}(a) & = r_T(a) \end{align*}nt^​(a)rt^​(a)​=nT​(a)=rT​(a)​

Thus, we have proved the following crucial property,

Lemma: Successive Elimination Key Property

For any suboptimal arm aaa (i.e., μ(a)<μ(a⋆)\mu(a) < \mu(a^{\star})μ(a)<μ(a⋆)), if it is still active at the end of round TTT, then,

Δ(a)≤O(rT(a))=O(log⁡(T)nT(a))\Delta(a) \leq \mathcal{O}(r_T(a)) = \mathcal{O}\left(\sqrt{\frac{\log(T)}{n_T(a)}}\right)Δ(a)≤O(rT​(a))=O(nT​(a)log(T)​​)

This is result is very nice, since it relates the suboptimality gap Δ(a)\Delta(a)Δ(a) to the number of times nT(a)n_T(a)nT​(a) that arm aaa has been played. Thus, an arm played many times must be close to optimal!

The rest of our analysis will rely on this result.

Regret Analysis

As per usual, when doing analysis we will construct the clean event and then connect it to the regret (the bad event will be negligible).

Let R(t;a)R(t; a)R(t;a) be the regret contribution of arm aaa at an arbitrary round ttt.

Thus,

R(t;a)=nt(a)⋅Δ(a)≤nt(a)⋅O(log⁡(T)nt(a))⏟O(log⁡(T)1/2nt(a)1/2)=O(nt(a)log⁡(T))\begin{align*} R(t; a) & = n_t(a) \cdot \Delta(a) \leq n_t(a) \cdot \underbrace{\mathcal{O}\left(\sqrt{\frac{\log(T)}{n_t(a)}}\right)}_{\mathcal{O}(\frac{\log(T)^{1/2}}{n_t(a)^{1/2}})} \newline & = \mathcal{O}\left(\sqrt{n_t(a) \log(T)}\right) \end{align*}R(t;a)​=nt​(a)⋅Δ(a)≤nt​(a)⋅O(nt​(a)1/2log(T)1/2​)O(nt​(a)log(T)​​)​​=O(nt​(a)log(T)​)​

We now introduce the set A+={a∣μ(a)<μ(a⋆)}\mathcal{A}^{+} = \{a \mid \mu(a) < \mu(a^{\star})\}A+={a∣μ(a)<μ(a⋆)} the set of all arms that contribute to the regret (i.e., all suboptimal arms).

Then, we can bound the total regret at round ttt as,

R(t)=∑a∈A+R(t;a)=∑a∈A+O(nt(a)log⁡(T))=O(log⁡(T))⏟does not depend on sum⋅∑a∈A+nt(a)≤O(log⁡(T))⋅∑a∈Ant(a)\begin{align*} R(t) & = \sum_{a \in \mathcal{A}^{+}} R(t; a) \newline & = \sum_{a \in \mathcal{A}^{+}} \mathcal{O}\left(\sqrt{n_t(a) \log(T)}\right) \newline & = \underbrace{\mathcal{O}(\sqrt{\log(T)})}_{\text{does not depend on sum}} \cdot \sum_{a \in \mathcal{A}^{+}} \sqrt{n_t(a)} \newline & \leq \mathcal{O}(\sqrt{\log(T)}) \cdot \sum_{a \in \mathcal{A}} \sqrt{n_t(a)} \end{align*}R(t)​=a∈A+∑​R(t;a)=a∈A+∑​O(nt​(a)log(T)​)=does not depend on sumO(log(T)​)​​⋅a∈A+∑​nt​(a)​≤O(log(T)​)⋅a∈A∑​nt​(a)​​

By Jensen’s inequality 1 1Here we will use the concave version which states, 1n∑i=1nf(xi)≤f(1n∑i=1nxi)\frac{1}{n} \sum_{i=1}^{n} f(x_i) \leq f\left(\frac{1}{n} \sum_{i=1}^{n} x_i\right)n1​∑i=1n​f(xi​)≤f(n1​∑i=1n​xi​) for a concave function fff, and the fact that ∑a∈Ant(a)=t\sum_{a \in \mathcal{A}} n_t(a) = t∑a∈A​nt​(a)=t, we have that,

1K∑a∈Ant(a)≤1K∑a∈Ant(a)=tK\frac{1}{K} \sum_{a \in \mathcal{A}} \sqrt{n_t(a)} \leq \sqrt{\frac{1}{K} \sum_{a \in \mathcal{A}} n_t(a)} = \sqrt{\frac{t}{K}}K1​a∈A∑​nt​(a)​≤K1​a∈A∑​nt​(a)​=Kt​​

Thus,

∑a∈Ant(a)≤KtK=Kt\sum_{a \in \mathcal{A}} \sqrt{n_t(a)} \leq K \sqrt{\frac{t}{K}} = \sqrt{K t}a∈A∑​nt​(a)​≤KKt​​=Kt​

Thus, we have our final regret bound,

[!THEOREM/Successive Elimination Regret Bound (K>2K > 2K>2)] For K>2K > 2K>2 arms, the successive elimination algorithm achieves the regret bound,

E[R(T)]≤O(Ktlog⁡(T))for all rounds t≤T\mathbb{E}[R(T)] \leq \mathcal{O}(\sqrt{K t \log(T)}) \quad \text{for all rounds } t \leq TE[R(T)]≤O(Ktlog(T)​)for all rounds t≤T

But there is another bound we can derive, from our previous property, one may rearrange to obtain,

Δ(a)≤O(log⁡(T)nT(a))  ⟹  nT(a)≤O(log⁡(T)(Δ(a))2)\Delta(a) \leq \mathcal{O}\left( \sqrt{\frac{\log(T)}{n_T(a)}}\right) \implies n_T(a) \leq \mathcal{O}\left(\frac{\log(T)}{(\Delta(a))^2}\right)Δ(a)≤O(nT​(a)log(T)​​)⟹nT​(a)≤O((Δ(a))2log(T)​)

If we now consider the set A+\mathcal{A}^{+}A+ of all suboptimal arms again, we can bound the regret for each arm a∈A+a \in \mathcal{A}^{+}a∈A+ as,

R(T;a)=Δ(a)⋅nT(a)≤Δ(a)⋅O(log⁡(T)(Δ(a))2)=O(log⁡(T)Δ(a))R(T; a) = \Delta(a) \cdot n_T(a) \leq \Delta(a) \cdot \mathcal{O}\left(\frac{\log(T)}{(\Delta(a))^2}\right) = \mathcal{O}\left(\frac{\log(T)}{\Delta(a)}\right)R(T;a)=Δ(a)⋅nT​(a)≤Δ(a)⋅O((Δ(a))2log(T)​)=O(Δ(a)log(T)​)

Summing over all suboptimal arms, we have,

R(T)≤O(log⁡(T))⋅∑a∈A+1Δ(a)R(T) \leq \mathcal{O}(\log(T)) \cdot \sum_{a \in \mathcal{A}^{+}} \frac{1}{\Delta(a)}R(T)≤O(log(T))⋅a∈A+∑​Δ(a)1​

[!THEOREM/Successive Elimination Regret Bound (K>2K > 2K>2)] For K>2K > 2K>2 arms, the successive elimination algorithm achieves the regret bound,

E[R(T)]≤O(log⁡(T))⋅∑a∈A+1Δ(a)\mathbb{E}[R(T)] \leq \mathcal{O}(\log(T)) \cdot \sum_{a \in \mathcal{A}^{+}} \frac{1}{\Delta(a)}E[R(T)]≤O(log(T))⋅a∈A+∑​Δ(a)1​

Instance-Dependent vs. Instance-Independent Bounds

We have now derived two different regret bounds for the successive elimination algorithm. The first one is an instance-independent bound, since it does not depend on the gaps Δ(a)\Delta(a)Δ(a), while the second one is instance-dependent, since it does depend on the gaps.

These bounds are lograithmic in TTT, with a constant that can be arbitrarily large (depending on the problem instance).

Thus, we conclude that adaptive exploration can lead to much better performance than uniform exploration (logarithmic vs. sublinear regret).

UCB and Optimism Under Uncertainty

Another adaptive exploration algorithm is the Upper Confidence Bound (UCB) algorithm.

The idea is to assume that each arm is as good as it can possiblyt be given the observations so far, and the agent chooses the best arm based on these optimistic estimates.

Summary: UCB Algorithm

Try each arm once;

In each round ttt, pick arg⁡max⁡a∈A UCBt(a)\underset{a \in \mathcal{A}}{\arg\max} \ \mathrm{UCB}_t(a)a∈Aargmax​ UCBt​(a), where UCBt(a)=μˉt(a)+rt(a)\mathrm{UCB}_t(a) = \bar{\mu}_t(a) + r_t(a)UCBt​(a)=μˉ​t​(a)+rt​(a).

But why does this UCB-based strategy work?

Imagine that an arm aaa is chosen in round ttt because it has a large UCBt(a)=μˉt(a)+rt(a)\mathrm{UCB}_t(a) = \bar{\mu}_t(a) + r_t(a)UCBt​(a)=μˉ​t​(a)+rt​(a).

Then, either,

  • The average reward μˉt(a)\bar{\mu}_t(a)μˉ​t​(a) is large   ⟹  \implies⟹ arm aaa is likely to have a high reward μ(a)\mu(a)μ(a),
  • and/or the confidence radius rt(a)r_t(a)rt​(a) is large   ⟹  \implies⟹ arm aaa has not been tried often, and thus we are uncertain about its true reward μ(a)\mu(a)μ(a).

Either reason makes the arm aaa a good candidate to try!

Note: Adaptive Exploration-Exploitation

Note that, not only is this UCB-based algorithm an adaptive exploration algorithm, it is also an adaptive exploration-exploitation algorithm.

The μˉt(a)\bar{\mu}_t(a)μˉ​t​(a) term encourages exploitation, while the rt(a)r_t(a)rt​(a) term encourages exploration, and summing them up (i.e., taking the UCB) is a natural way to trade-off exploration and exploitation adaptively.

Analysis

Recall the clean event (from reward tape idea),

E≔{∀a∀t∣μˉt(a)−μ(a)∣≤rt(a)}.\mathcal{E} \coloneqq \{\forall a \forall t |\bar{\mu}_t(a) - \mu(a) | \leq r_t(a) \}.E:={∀a∀t∣μˉ​t​(a)−μ(a)∣≤rt​(a)}.

Assume a⋆a^{\star}a⋆ be an optimal arm and ata_tat​ is the arm chosen by the agent at round ttt, i.e.,

UCBt(at)≥UCBt(a⋆)\mathrm{UCB}_t(a_t) \geq \mathrm{UCB}_t(a^{\star})UCBt​(at​)≥UCBt​(a⋆)

We know that, under the clean event,

μ(at)+rt(at)≥μˉt(at)\mu(a_t) + r_t(a_t) \geq \bar{\mu}_t(a_t)μ(at​)+rt​(at​)≥μˉ​t​(at​)

By definition of UCB and combining the two inequalities, we have,

UCBt(a⋆)≥μ(a⋆)\mathrm{UCB}_t(a^{\star}) \geq \mu(a^{\star})UCBt​(a⋆)≥μ(a⋆)

Then,

μ(at)+2rt(at)≥μˉt(at)+rt(at)μ(at)+2rt(at)−rt(at)≥μˉt(at)+rt(at)−rt(at)μ(at)+rt(at)⏟UCBt(at)≥μˉt(a⋆)UCBt(at)≥UCBt(a⋆)⏟by our assumption≥μˉt(a⋆)\begin{align*} \mu(a_t) + 2 r_t(a_t) & \geq \bar{\mu}_t(a_t) + r_t(a_t) \newline \mu(a_t) + 2 r_t(a_t) - r_t(a_t) & \geq \bar{\mu}_t(a_t) + r_t(a_t) - r_t(a_t) \newline \underbrace{\mu(a_t) + r_t(a_t)}_{\mathrm{UCB}_t(a_t)} & \geq \bar{\mu}_t(a^{\star}) \newline \mathrm{UCB}_t(a_t) & \underbrace{\geq \mathrm{UCB}_t(a^{\star})}_{\text{by our assumption}} \geq \bar{\mu}_t(a^{\star}) \newline \end{align*}μ(at​)+2rt​(at​)μ(at​)+2rt​(at​)−rt​(at​)UCBt​(at​)μ(at​)+rt​(at​)​​UCBt​(at​)​≥μˉ​t​(at​)+rt​(at​)≥μˉ​t​(at​)+rt​(at​)−rt​(at​)≥μˉ​t​(a⋆)by our assumption≥UCBt​(a⋆)​​≥μˉ​t​(a⋆)​

Using this chain of inequalities, we find that,

μ(at)+2rt(at)≥μ(a⋆),\mu(a_t) + 2 r_t(a_t) \geq \mu(a^{\star}),μ(at​)+2rt​(at​)≥μ(a⋆),

Rearranging this we have found an expression of the gap Δ(at)\Delta(a_t)Δ(at​),

Δ(at):−μ(a⋆)−μ(at)≤2rt(at)=22log⁡(T)nt(at)\Delta(a_t) \coloneq \mu(a^{\star}) - \mu(a_t) \leq 2 r_t(a_t) = 2 \sqrt{\frac{2 \log(T)}{n_t(a_t)}}Δ(at​):−μ(a⋆)−μ(at​)≤2rt​(at​)=2nt​(at​)2log(T)​​

Now (again), for each arm aaa, consider the last round t^\hat{t}t^ when arm aaa was chosen (or round TTT if it was chosen in the last round) by the agent,

Δ(a)≤O(log⁡(T)nt^(a))=O(log⁡(T)nT(a))\Delta(a) \leq \mathcal{O}\left(\sqrt{\frac{\log(T)}{n_{\hat{t}}(a)}}\right) = \mathcal{O}\left(\sqrt{\frac{\log(T)}{n_T(a)}}\right)Δ(a)≤O(nt^​(a)log(T)​​)=O(nT​(a)log(T)​​)

Thus, we have proved the same key property as for successive elimination!

One can use the exact same analysis as for successive elimination to obtain the same two regret bounds.

E[R(T)]≤O(Ktlog⁡(T))for all rounds t≤T\mathbb{E}[R(T)] \leq \mathcal{O}(\sqrt{K t \log(T)}) \quad \text{for all rounds } t \leq TE[R(T)]≤O(Ktlog(T)​)for all rounds t≤T E[R(T)]≤O(log⁡(T))⋅∑a∈A+1Δ(a)\mathbb{E}[R(T)] \leq \mathcal{O}(\log(T)) \cdot \sum_{a \in \mathcal{A}^{+}} \frac{1}{\Delta(a)}E[R(T)]≤O(log(T))⋅a∈A+∑​Δ(a)1​

Lower Bound

We have so far only discussed upper bounds on the regret. But what about lower bounds?

The performance of any algorithm is determined by the distance between the optimal arm and other arms. But, hard problems have arms with similar distributions.

These distances are formally described by the gaps Δ(a)\Delta(a)Δ(a) and the distribution distances KL(Da,Da⋆)\mathrm{KL}(\mathcal{D}_a, \mathcal{D}_{a^{\star}})KL(Da​,Da⋆​) 2Here, KL(⋅,⋅)\mathrm{KL}(\cdot, \cdot)KL(⋅,⋅) is the Kullback-Leibler divergence between two distributions, ∑x∈XP(x)log⁡(P(x)Q(x))\sum_{x \in \mathcal{X}} P(x) \log\left(\frac{P(x)}{Q(x)}\right) ∑x∈X​P(x)log(Q(x)P(x)​) .

We will end this part of the course wit the Lai and Robbins lower bound theorem,

Theorem: Lai and Robbins Lower Bound

The (asymptotic) total regret is at least logarithmic in number of steps,
