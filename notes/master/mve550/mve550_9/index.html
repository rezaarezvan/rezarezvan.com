Introduction

In this part, we will introduce Continuous-Time Markov Chains, which are stochastic processes that evolve over continuous time and possess the Markov property. They are widely used in various fields, including queueing theory, reliability engineering, and population dynamics.

Continuous-Time Markov Chains

Definition: Continuous-Time Markov Chain

A continuous-time stochastic process {Xt}t≥0\{X_t\}_{t \geq 0}{Xt​}t≥0​ with discrete state space S\mathcal{S}S is a continuous-time Markov chain if,

P(Xt+s=j∣Xs=i,Xu,0≤u<s)=P(Xt+s=j∣Xs=i),P(X_{t + s} = j \mid X_s = i, X_u, 0 \leq u < s) = P(X_{t + s} = j \mid X_s = i),P(Xt+s​=j∣Xs​=i,Xu​,0≤u<s)=P(Xt+s​=j∣Xs​=i),

where s,t≥0s, t \geq 0s,t≥0 and i,j,Xu∈Si, j, X_u \in \mathcal{S}i,j,Xu​∈S.

The process is time-homogeneous if for s,t≥0s, t \geq 0s,t≥0 and all i,j∈Si, j \in \mathcal{S}i,j∈S,

P(Xt+s=j∣Xs=i)=P(Xt=j∣X0=i).P(X_{t + s} = j \mid X_s = i) = P(X_t = j \mid X_0 = i).P(Xt+s​=j∣Xs​=i)=P(Xt​=j∣X0​=i).

We then define the transition function as the matrix function P(t)P(t)P(t) with entries of the matrix given by,

P(t)ij=P(Xt=j∣X0=i).P(t)_{ij} = P(X_t = j \mid X_0 = i).P(t)ij​=P(Xt​=j∣X0​=i).

The Chapman-Kolmogorov Equations

Theorem: Chapman-Kolmogorov Equations

For the transition function P(t)P(t)P(t) we have,

  • P(s+t)=P(s)P(t)P(s + t) = P(s) P(t)P(s+t)=P(s)P(t)
  • P(0)=IP(0) = IP(0)=I (identity matrix)
Proof

For the first property; If they are same for an arbitrary element i,ji, ji,j of the matrix, then they are equal.

P(s+t)ij=P(Xs+t=j∣Xs=i)=∑kP(Xs+t=j,Xs=k∣X0=i)=∑kP(Xs=k∣X0=i)P(Xs+t=j∣Xs=k,X0=i)=∑kP(Xs=k∣X0=i)P(Xs+t=j∣Xs=k)=∑kP(Xs=k∣X0=i)P(Xt=j∣X0=k)=∑kP(s)ikP(t)kj=(P(s)P(t))ij ■\begin{align*} P(s + t)_{ij} & = P(X_{s + t} = j \mid X_s = i) \newline & = \sum_k P(X_{s + t} = j, X_s = k \mid X_0 = i) \newline & = \sum_k P(X_s = k \mid X_0 = i) P(X_{s + t} = j \mid X_s = k, X_0 = i) \newline & = \sum_k P(X_s = k \mid X_0 = i) P(X_{s + t} = j \mid X_s = k) \newline & = \sum_k P(X_s = k \mid X_0 = i) P(X_t = j \mid X_0 = k) \newline & = \sum_k P(s)_{ik} P(t)_{kj} \newline & = (P(s) P(t))_{ij} \ _\blacksquare \end{align*}P(s+t)ij​​=P(Xs+t​=j∣Xs​=i)=k∑​P(Xs+t​=j,Xs​=k∣X0​=i)=k∑​P(Xs​=k∣X0​=i)P(Xs+t​=j∣Xs​=k,X0​=i)=k∑​P(Xs​=k∣X0​=i)P(Xs+t​=j∣Xs​=k)=k∑​P(Xs​=k∣X0​=i)P(Xt​=j∣X0​=k)=k∑​P(s)ik​P(t)kj​=(P(s)P(t))ij​ ■​​

The second property follows directly from the definition of the transition function.

Holding Times

Define TiT_iTi​ as the time the continuous-time Markov chain {Xt}t≥0\{X_t\}_{t \geq 0}{Xt​}t≥0​ that always starts in state iii stays in iii before moving to a different state, so that for any s>0s > 0s>0,

The Embedded Markov Chain

Intuition: Embedded Markov Chain

We can define a new stochastic process by listing the states the chain visits. This will be a discrete time Markov chain called the embedded Markov chain with transition matrix P~\tilde{P}P~.

Note

The transition matrix P~\tilde{P}P~ has zeros along its diagonal! Why? Because the process cannot stay in the same state when it makes a transition.

Further, note that the continuous-time Markov chain is completely determined by the expected holding times (1q1,…,1qk)(\frac{1}{q_1}, \ldots, \frac{1}{q_k})(q1​1​,…,qk​1​) and the transition matrix of the embedded Markov chain P~\tilde{P}P~.

Intuition: Constructing a Continuous-Time Markov Chain

A way to describe a continuous-time Markov chain is to describe k×(k−1)k \times (k - 1)k×(k−1) independent “alarm clocks”.

For states iii and jjj so that i≠ji \neq ji=j, let qijq_{ij}qij​ be the parameter of an Exponentially distributed random variable representing the time until an “alarm clock” rings.

When in state iii, wait until the first alarm clock qiq_iqi​ rings, then move to the state given by the index jjj of that alarm clock. This defines a continuous-time Markov chain.

The time until the first alarm clock qiq_iqi​ rings is Exponentially distributed with parameter given by,

qi=qi1+qi2+…+qi,i−1+qi,i+1+…+qik.q_i = q_{i1} + q_{i2} + \ldots + q_{i, i - 1} + q_{i, i + 1} + \ldots + q_{ik}.qi​=qi1​+qi2​+…+qi,i−1​+qi,i+1​+…+qik​.

i.e., the parameter of the holding time distribution at iii.

The chain is completely described by the rates qijq_{ij}qij​.

Further, we saw also that the chain is completely described by the pijp_{ij}pij​ and the qiq_iqi​. The relationship is described by the equation above, and,

pij=qijqi1+qi2+…+qi,i−1+qi,i+1+…+qik=qijqi.p_{ij} = \frac{q_{ij}}{q_{i1} + q_{i2} + \ldots + q_{i, i - 1} + q_{i, i + 1} + \ldots + q_{ik}} = \frac{q_{ij}}{q_i}.pij​=qi1​+qi2​+…+qi,i−1​+qi,i+1​+…+qik​qij​​=qi​qij​​.

for i≠ji \neq ji=j.

Kolmogorov Forward and Backward Equations

Theorem: Kolmogorov Forward and Backward Equations

The Kolmogorov Forward and Backward equations describe how the transition function P(t)P(t)P(t) evolves over time.

We get that for all t≥0t \geq 0t≥0,

P′(t)=P(t)Q=QP(t).P^{\prime}(t) = P(t) Q = Q P(t).P′(t)=P(t)Q=QP(t).

Thus, what this means in terms of the components of P(t)P(t)P(t),

P′(t)ij=−Pij(t)qj+∑k≠jPik(t)qkjP′(t)ij=−qiPij(t)+∑k≠iqikPkj(t)\begin{align*} P^{\prime}(t)_{ij} & = -P_{ij}(t) q_j + \sum_{k \neq j} P_{ik}(t) q_{kj} \newline P^{\prime}(t)_{ij} & = -q_i P_{ij}(t) + \sum_{k \neq i} q_{ik} P_{kj}(t) \newline \end{align*}P′(t)ij​P′(t)ij​​=−Pij​(t)qj​+k=j∑​Pik​(t)qkj​=−qi​Pij​(t)+k=i∑​qik​Pkj​(t)​

Thus, the above equations define a set of differential equations which the components of the matrix function P(t)P(t)P(t) must satisfy.

Proof
P′(t)=lim⁡h→0P(t+h)−P(t)h=lim⁡h→0P(t)P(h)−P(t)h=lim⁡h→0P(t)P(h)−Ih=P(t)lim⁡h→0P(h)−Ih=P(t)lim⁡h→0P(h)−P(0)h⏟P′(0)=Q=P(t)Q ■\begin{align*} P^{\prime}(t) & = \lim_{h \to 0} \frac{P(t + h) - P(t)}{h} \newline & = \lim_{h \to 0} \frac{P(t) P(h) - P(t)}{h} \newline & = \lim_{h \to 0} P(t) \frac{P(h) - I}{h} \newline & = P(t) \lim_{h \to 0} \frac{P(h) - I}{h} \newline & = P(t) \underbrace{\lim_{h \to 0} \frac{P(h) - P(0)}{h}}_{P^{\prime}(0) = Q} \newline & = P(t) Q \ _\blacksquare \end{align*}P′(t)​=h→0lim​hP(t+h)−P(t)​=h→0lim​hP(t)P(h)−P(t)​=h→0lim​P(t)hP(h)−I​=P(t)h→0lim​hP(h)−I​=P(t)P′(0)=Qh→0lim​hP(h)−P(0)​​​=P(t)Q ■​​

The Matrix Exponential

Intuition: Matrix Exponential Solution

The solution to the Kolmogorov Forward and Backward equations is given by the matrix exponential. For any square matrix AAA we can define the matrix exponential as,

eA≔∑n=0∞1n!An=I+A+12A2+16A3+…e^A \coloneqq \sum_{n = 0}^{\infty} \frac{1}{n!} A^n = I + A + \frac{1}{2} A^2 + \frac{1}{6} A^3 + \ldotseA:=n=0∑∞​n!1​An=I+A+21​A2+61​A3+…

The series converges for all square matrices AAA (but we will not prove this here).

Some important properties of the matrix exponential are:

  • e0=Ie^0 = Ie0=I
  • eAe−A=Ie^A e^{-A} = IeAe−A=I
  • e(s+t)A=esAetAe^{(s + t) A} = e^{sA} e^{tA}e(s+t)A=esAetA
  • If AB=BAAB = BAAB=BA, then eA+B=eAeB=eBeAe^{A + B} = e^A e^B = e^B e^AeA+B=eAeB=eBeA
  • ∂∂tetA=AetA=etAA\frac{\partial}{\partial t} e^{tA} = A e^{tA} = e^{tA} A∂t∂​etA=AetA=etAA

Note that P(t)=etQP(t) = e^{tQ}P(t)=etQ is the unique solution to the differential equations given by the Kolmogorov Forward and Backward equations with initial condition P(0)=IP(0) = IP(0)=I.

Further, note that P′(t)=QP(t)P^\prime(t) = Q P(t)P′(t)=QP(t) for all t≥0t \geq 0t≥0 and P(0)=IP(0) = IP(0)=I.
