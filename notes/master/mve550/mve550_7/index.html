Introduction

In this part we will introduce Poisson processes and their properties.

Poisson Processes

Intuition: Poisson Processes

In the beginning, we defined a stochastic process as a collection {Xt,t∈I}\{X_t, t \in \mathcal{I}\}{Xt​,t∈I} of random variable with a common state space S\mathcal{S}S. Initially, the state space was discrete (i.e., finite or countable), last time we extended this to continuous state spaces.

In this part, we will extend even further and make the set I\mathcal{I}I is a non-countable set, for example, all positive real numbers, or all subsets of R2\mathbb{R}^2R2.

Intuition: Poisson Process

A random variable with valuyes 0,1,2,…0, 1, 2, \ldots0,1,2,… with a Poisson distribution can be used to model the count of events happening independently, e.g., within some time interval.

We know that if X∼Poisson(λ)X \sim \mathrm{Poisson}(\lambda)X∼Poisson(λ), then,

π(x)=e−λλxx!,\pi(x) = e^{-\lambda} \frac{\lambda^x}{x!},π(x)=e−λx!λx​,

and,

E[X]=λ,Var(X)=λ.\mathbb{E}[X] = \lambda, \quad \mathrm{Var}(X) = \lambda.E[X]=λ,Var(X)=λ.

A Poisson process models not only the count for a specific time interval, but also the exact time of every event.

Exponential Distribution and Poisson Processes

Recall: Exponential Distribution

A random variable XXX with non-negative real values as possible values has an exponential distribution with parameter λ\lambdaλ if the density is,

π(x)≔λe−λx.\pi(x) \coloneqq \lambda e^{-\lambda x}.π(x):=λe−λx.

The cumulative probability distribution is,

F(x)≔1−e−λx.F(x) \coloneqq 1 - e^{-\lambda x}.F(x):=1−e−λx.

The expectation is and variance are,

E[X]=1λ,Var(X)=1λ2.\mathbb{E}[X] = \frac{1}{\lambda}, \quad \mathrm{Var}(X) = \frac{1}{\lambda^2}.E[X]=λ1​,Var(X)=λ21​.
Note: Memoryless of the Exponential Distribution

A random variable XXX is called memoryless if,

P(X>s+t∣X>s)=P(X>t)P(X > s + t \mid X > s) = P(X > t)P(X>s+t∣X>s)=P(X>t)

for all s>0,t>0s > 0, t > 0s>0,t>0.

The exponential distribution is memoryless, and is the only memoryless continuous random variable. Thus, we need to take this into consideration when modeling waiting times.
