Introduction

In this part, we will look at parameter inference for discrete-time finite state space Markov chains. We will start by looking at the multinomial-Dirichlet conjugacy, which can be seen as a generalization of the binomial-Beta conjugacy we saw earlier.

Further, we will look att Hidden Markov Models and inference for these. Then we will look at inference for branching processes, Poisson processes, and finally continuous-time finite state space Markov chains.

Hidden Markov Models (HMMs)

Definition 3 (Hidden Markov Model)

A Hidden Markov Model (HMMs) consists of:

  • A Markov chain X0,…,Xn,…X_0, \ldots, X_n, \ldotsX0​,…,Xn​,… and,
  • another sequence Y0,…,Yn,…Y_0, \ldots, Y_n, \ldotsY0​,…,Yn​,… such that,
P(Yk∣Y0,…,Yk−1,X0,…,Xn)=P(Yk∣Xk).P(Y_k \mid Y_0, \ldots, Y_{k - 1}, X_0, \ldots, X_n) = P(Y_k \mid X_k).P(Yk​∣Y0​,…,Yk−1​,X0​,…,Xn​)=P(Yk​∣Xk​).

In some models we may instead have,

P(Yk∣Y0,…,Yk−1,X0,…,Xn)=P(Yk∣Yk−1,Xk).P(Y_k \mid Y_0, \ldots, Y_{k - 1}, X_0, \ldots, X_n) = P(Y_k \mid Y_{k - 1}, X_k).P(Yk​∣Y0​,…,Yk−1​,X0​,…,Xn​)=P(Yk​∣Yk−1​,Xk​).

Generally, Y0,…,YnY_0, \ldots, Y_nY0​,…,Yn​ are called observations and X0,…,XnX_0, \ldots, X_nX0​,…,Xn​ are called hidden states. The XiX_iXi​’s represent the “underlying process” that the observed values YiY_iYi​’s depend on. Further, generally the XkX_kXk​ have a finite state space.

Intuition (Inference in HMMs)

When the parameters of the HMM are known, we want to know about the values of the hidden variables XiX_iXi​, for example,

  • What is the most likely sequence X0,X1,…,XnX_0, X_1, \ldots, X_nX0​,X1​,…,Xn​ given the data?
  • What is the probability distribution for a single XiX_iXi​ given the data? However, when the parameters of the HMM are unknown, we need to infer these from some data.

If data with all XiX_iXi​ and YiY_iYi​ known is available, inference for parameters is based on counts of transitions. The inference for the Markov chain is exactly as for the Markov chains we have looked at before. The inference for the emission probabilities, i.e., the parameters of P(Yk∣Xk)P(Y_k \mid X_k)P(Yk​∣Xk​), can be done independently of the inference for the Markov chain.

Bayesian Inference for Branching Processes

Intuition (Bayesian Inference for Galton-Watson Branching Processes)

Say you have observed some data, and you want to find a Galton-Watson branching process that appropriately models the data, to then make predictions about future observations.

Recall that a branching process is characterized by the probability vector a=(a0,a1,a2,…)a = (a_0, a_1, a_2, \ldots)a=(a0​,a1​,a2​,…), where aia_iai​ is the probability for iii offspring in the offspring process.

Let y1,y2,…,yny_1, y_2, \ldots, y_ny1​,y2​,…,yn​ be the counts of offspring in nnn observations of the offspring process. If aaa is given, we have the likelihood,

π(y1,y2,…,yn∣a)≔∏i=1nayi.\pi(y_1, y_2, \ldots, y_n \mid a) \coloneqq \prod_{i = 1}^n a_{y_i}.π(y1​,y2​,…,yn​∣a):=i=1∏n​ayi​​.

Thus, to complete the model, we need a prior on aaa.

Since aaa has an infinite length, and we have a finite number of observations, we need to put information from the context into the prior, to get a sensible posterior.

Assume the offspring process is Binomial(N,p)\mathrm{Binomial}(N, p)Binomial(N,p) for some parameter ppp and a fixed (known) NNN. By definition, we get the likelihood,

π(y1,y2,…,yn∣p)≔∏i=1nBinomial(yi;N,p)\pi(y_1, y_2, \ldots, y_n \mid p) \coloneqq \prod_{i = 1}^n \mathrm{Binomial}(y_i; N, p)π(y1​,y2​,…,yn​∣p):=i=1∏n​Binomial(yi​;N,p)

Bayesian Inference for Poisson Processes

For a homogeneous Poisson process, we setup up a prior π(λ)\pi(\lambda)π(λ) for its parameter λ\lambdaλ, and find the posterior given the observations.

Poisson(y;λt)=exp⁡(−λt)(λt)yy!∝λexp⁡(−λt)λy.\mathrm{Poisson}(y; \lambda t) = \exp(-\lambda t) \frac{(\lambda t)^y}{y!} \propto_{\lambda} \exp(-\lambda t) \lambda^y.Poisson(y;λt)=exp(−λt)y!(λt)y​∝λ​exp(−λt)λy.

Bayesian Inference for Continuous-Time Markov Chains

Intuition (Bayesian Inference for Continuous-Time Markov Chains)

Lastyl, recall that a continuous-time Markov chain with finite state space is completely characterized by its holding times parameter vector qqq and the transition matrix P~\tilde{P}P~ of its embedded Markov chain.

Parametrizing instead with the “alarm clock” parameters qijq_{ij}qij​ gives an equivalent description of the process.

The two parts of the data can be considered independently are,

  • We learn about P~\tilde{P}P~ from the counts of transitions between states, and
  • We learn about qqq from the observed lengths of stays the process has in each state.

For P~\tilde{P}P~ the situation is analogous to the one for discrete-time Markov chains, except that the diagonal of P~\tilde{P}P~ must be zero, so the prior must exclude the possibility of non-zero diagonal elements.

For example, for P~1\tilde{P}_1P~1​, the first row of P~\tilde{P}P~, we might use the prior Dirichlet(0,1,…,1)\mathrm{Dirichlet}(0, 1, \ldots, 1)Dirichlet(0,1,…,1), i.e., a Dirichlet prior with a zero pseudocount for the first element.

The holding times in state iii are distributed as Exponential(qi)\mathrm{Exponential}(q_i)Exponential(qi​). If we have observed a total holding time of hhh over nnn intervals, that data has likelihood proportional to e−hqiqine^{-hq_i}q_i^ne−hqi​qin​. Using qi∼Gamma(α,β)q_i \sim \mathrm{Gamma}(\alpha, \beta)qi​∼Gamma(α,β) as prior results in the posterior qi∣D∼Gamma(α+n,β+h)q_i \mid \mathcal{D} \sim \mathrm{Gamma}(\alpha + n, \beta + h)qi​∣D∼Gamma(α+n,β+h).
