Introduction to Stochastic Processes

In this part, we will define what a stochastic process. We will further recap some basic probability theory concepts that are essential for understanding stochastic processes.

In Figure 1, we can see that some models make exact predictions (deterministic processes), while others provide predictions with uncertainty (stochastic processes). In most cases, it is more realistic to model systems as stochastic processes, as they often involve inherent randomness and uncertainty.

Note

We will use the terminology stochastic models and probabilistic models interchangeably.

Definition: Stochastic Process

A stochastic process is a collection of random variables, {Xt,t∈I}\{X_t, t \in \mathcal{I} \}{Xt​,t∈I}.

The set I\mathcal{I}I is the index set of the process. I\mathcal{I}I most often represents a set of specific times.

The random variables XtX_tXt​ are defined on a common state space S\mathcal{S}S. This set represents the possible values the random variables XtX_tXt​ can take.

Finally, these sets can be discrete or continuous and simple (e.g., I={1,2,3}\mathcal{I} = \{1, 2, 3\}I={1,2,3}) or more complex (e.g., vectors of real numbers).

Motivation for Probabilistic Models

One thing we need to consider is, what do we want to do with these models? What is the point? While it is an open question, a easy answer is that we want to set up a model, based on some general knowledge, and make new predictions from the model.

  1. You have data.
  2. You want to find a model so that the data could reasonably be produced by the model.
  3. You want to use this model for predictions of future observations.

Using data in this way is called (statistical) inference.

Definition: Frequentist VS. Bayesian Inference

There are two main paradigms for statistical inference: frequentist inference and Bayesian inference.

In classical (frequentist) inference,

  1. We use contextual knowledge to find a model with some unknown parameters. The model relates observed data to future predictions.

  2. Find an estimate for the parameters, using estimators that have desirable properties.

  3. Plug the estimates into the model and make predictions.

In Bayesian inference,

  1. We use contextual knowledge to find a model with some uncertain parameters. The model relates observed data to future predictions.

  2. Find the conditional probability for future predictions given the value of the observed data.

The Markov Property

Definition: Markov Property

A process fulfills the Markov property if, for any t0∈It_0 \in \mathcal{I}t0​∈I, whenever Xt0X_{t_0}Xt0​​ is known, XtX_tXt​ (with t>t0t > t_0t>t0​) is independent of the values for XsX_sXs​ for all s<t0s < t_0s<t0​.

Note

Most of the stochastic processes that we will deal with will have the Markov property.

Probability Theory Recap

Intuition: Random Variable

A random variable is a variable which has possible values in some state space S\mathcal{S}S, together with probabilities assigned to values and sets of values in the state space.

We separate between discrete and continuous random variables.

For discrete random variables, we assign a probability to each single value in the state space.

For continuous random variables, we assign probabilities to intervals of values in the state space.

Definition: Measurable Subsets

It turns out to be impossible to define a concept with reasonable properties that assigns a “size” to all subsets of for example RN\mathbb{R}^NRN. Instead, we need to restrict the concept of “size” to “measurable” sets.

Let SSS be any set. A sigma-algebra Ω\OmegaΩ on SSS is a set of subsets of SSS such that,

  1. Ω\OmegaΩ includes SSS.

  2. If A∈ΩA \in \OmegaA∈Ω, then the complement Ac≔S∖AA^c \coloneqq S \setminus AAc:=S∖A is also in Ω\OmegaΩ.

  3. If A1,A2,…∈ΩA_1, A_2, \ldots \in \OmegaA1​,A2​,…∈Ω, then the union ⋃i=1∞Ai\bigcup_{i=1}^{\infty} A_i⋃i=1∞​Ai​ is also in Ω\OmegaΩ.

Measurable sets are those that are in an appropriately defined sigma-algebra.

Note

In RN\mathbb{R}^NRN, one generally uses the Borel subsets 1

What is most important, when SSS is finite or countable, all subsets will be measurable. When SSS is some interval of real numbers, there will exist subsets that are not measurable, but we will not be concerned with those.

Definition: Probability Measure and Random Variables

A probability (measure) is a real function P(⋅)P(\cdot)P(⋅) defined on the measureable subsets A⊆SA \subseteq SA⊆S, satisfying,

  1. 0≤P(A)≤10 \leq P(A) \leq 10≤P(A)≤1 for all measurable subsets A⊆SA \subseteq SA⊆S.

  2. P(S)=1P(S) = 1P(S)=1.

  3. For any countable collection of disjoint measurable subsets A1,A2,…A_1, A_2, \ldotsA1​,A2​,…, we have P(⋃i=1∞Ai)=∑i=1∞P(Ai)P\left( \bigcup_{i=1}^{\infty} A_i \right) = \sum_{i=1}^{\infty} P(A_i)P(⋃i=1∞​Ai​)=∑i=1∞​P(Ai​).

These are called the Kolmogorov axioms for probability 2.

Measurable subsets SSS are called events, and P(S)P(S)P(S) is the probability of the event SSS occurring.

Thus, a random variable XXX with state space S\mathcal{S}S is a real-valued (measurable) function on SSS together with a probability measure P(⋅)P(\cdot)P(⋅) defined on the measurable subsets of S\mathcal{S}S.

Recall: Conditional Probability and Independence

Given the events AAA and BBB, the conditional probability of AAA given BBB is defined as,

P(A∣B)=P(A∩B)P(B)P(A \mid B) = \frac{P(A \cap B)}{P(B)}P(A∣B)=P(B)P(A∩B)​

The events AAA and BBB are independent if P(A∩B)=P(A)P(B)P(A \cap B) = P(A) P(B)P(A∩B)=P(A)P(B), or equivalently, if P(A∣B)=P(A)P(A \mid B) = P(A)P(A∣B)=P(A).

The law of total probability states that if B1,B2,…,BnB_1, B_2, \ldots, B_nB1​,B2​,…,Bn​ is a sequence of events that partitions S. Then,

P(A)=∑i=1nP(A∩Bi)=∑i=1nP(A∣Bi)P(Bi)P(A) = \sum_{i=1}^{n} P(A \cap B_i) = \sum_{i=1}^{n} P(A \mid B_i) P(B_i)P(A)=i=1∑n​P(A∩Bi​)=i=1∑n​P(A∣Bi​)P(Bi​)

Thus, Bayes’ law follow directly from the definition of conditional probability and the law of total probability,

P(B∣A)=P(A∣B)P(B)P(A)P(B \mid A) = \frac{P(A \mid B) P(B)}{P(A)}P(B∣A)=P(A)P(A∣B)P(B)​

Definition: Conditional Densities for Continuous Distributions

For a continuous random variable XXX, we will write its density function as π(x)\pi(x)π(x), extending the generic π\piπ-notation. If we have a joint distribution for continuous random variables XXX and YYY, we write the joint density as π(x,y)\pi(x, y)π(x,y).
