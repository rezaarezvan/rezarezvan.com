Introduction

In this part, we will introduce and define Markov Chains along with some examples. We will conduct some basic computations using them and investigate their long term evolution using powers of matrices. Finally, we will discuss induction and limiting and stationary distributions.

Markov Chains

Definition: Markov Chain

Let SSS be a discrete set (not necessarily finite), called the state space. A Markov Chain is a sequence of random variables X0,X1,X2,…X_0, X_1, X_2, \ldotsX0​,X1​,X2​,… taking values in SSS, with the property,

π(Xn+1∣X0,X1,…,Xn)=π(Xn+1∣Xn)\pi(X_{n+1} \mid X_0, X_1, \ldots, X_n) = \pi(X_{n+1} \mid X_n)π(Xn+1​∣X0​,X1​,…,Xn​)=π(Xn+1​∣Xn​)

for all n≥1n \geq 1n≥1.

The chain is said to be time-homogeneous if, for all n≥0n \geq 0n≥0 1We generally assume this unless otherwise stated.,

π(Xn+1∣Xn)=π(X1∣X0)\pi(X_{n+1} \mid X_n) = \pi(X_1 \mid X_0)π(Xn+1​∣Xn​)=π(X1​∣X0​)

The transition matrix is defined with,

Pij=π(Xn+1=j∣Xn=i)P_{ij} = \pi(X_{n+1} = j \mid X_n = i)Pij​=π(Xn+1​=j∣Xn​=i)

Further, a stochastic matrix is a square matrix PPP with non-negative entries, satisfying P1=1P \mathbf{1} = \mathbf{1}P1=1, where 1\mathbf{1}1 is a row vector of ones (i.e., all rows sum to one).

Lastly, all transition matrices are stochastic matrices, and all stochastic matrices can be used as transition matrices.

Induction, Limiting and Stationary Distributions

Intuition: Proving stuff using Induction

Many results about Markov chains can be proved using induction.

  1. Formulate a statement S(n)S(n)S(n) depending on a non-negative integer nnn.

  2. Prove S(0)S(0)S(0) (the base case).

  3. Prove that if S(n)S(n)S(n) is true, then S(n+1)S(n + 1)S(n+1) is true (the inductive step).

With this, one may conclude that S(n)S(n)S(n) is true for all non-negative integers nnn.

Definition: Limiting Distribution

A limiting distribution for a Markov chain with transition matrix PPP is a probability vector vvv such that,

lim⁡n→∞(Pn)ij=vj\lim_{n \to \infty} (P^n)_{ij} = v_jn→∞lim​(Pn)ij​=vj​

for all states iii and jjj.

An equivalent formulation is, the limit lim⁡n→∞(Pn)ij\lim_{n \to \infty} (P^n)_{ij}limn→∞​(Pn)ij​ exists and does not depend on iii.

Another equivalent formulation is, lim⁡n→∞Pn\lim_{n \to \infty} P^nlimn→∞​Pn is a stochastic matrix with all rows identical.

Further, a Markov chain has either no or one unique limiting distribution. If a limiting distribution exists, its probabilities correspond to the proportion of time steps the chain spends at each state in the long run.

Definition: Stationary Distribution

A stationary distribution for a Markov chain is a distribution that is unchanged when applying one step of the Markov chain. If PPP is the transition matrix, then a probability vector vvv represents a stationary distribution if and only if,

vP=vvP = vvP=v

A Markov chain can have zero, one, or many stationary distributions. Limiting distributions are always stationary distributions (but not necessarily vice versa).

Definition: Regular Transition Matrices

A stochastic matrix PPP is positive if all entries are strictly positive, i.e., Pij>0P_{ij} > 0Pij​>0 for all iii and jjj. A stochastic matrix PPP is regular if PnP^nPn is positive for some n>0n > 0n>0.

Theorem: Limit Theorem for Regular Markov Chains

If the transition matrix PPP is regular, the limiting distribution exists. There are no other stationary distributions and the limiting distribution is positive, i.e., all states have positive probability in the limiting distribution.

Intuition: Finding Stationary Distributions

Find the vvv satisfying vP=vvP = vvP=v by, for example,

  1. Solving the linear system vP=vvP = vvP=v

  2. Guessing at a vvv, and showing that vP=vvP = vvP=v, or,

  3. Computing an eigenvector for the transpose PTP^TPT belonging to the eigenvalue 1.

Having found a vvv satisfying vP=vvP = vvP=v, if the transition matrix PPP is regular, we know vvv represents the unique limiting distribution and the unique stationary distribution.

Example: Random Walks on Undirected Graphs

An undirected graph consists of nodes and undirected edges connecting them. An undirected graph defines a random walk Markov chain by, at every time step, following one of the edges out of a node, with equal probability.

When the graph is finite, we show that the vector uuu is a stationary distribution, where ui=deg(i)Su_i = \frac{\mathrm{deg}(i)}{S}ui​=Sdeg(i)​, where deg(i)\mathrm{deg}(i)deg(i) is the number of edges going into node iii and SSS is the sum over all nodes of their degrees.

If we define the Markov chain by choosing the next node with probabilities according to the weights. Intuition: Moving Between States

State jjj is accessible from state iii if (Pn)ij>0(P^n)_{ij} > 0(Pn)ij​>0 for some n≥0n \geq 0n≥0. States iii and jjj communicate if iii is accessible from jjj and jjj is accessible from iii.

“Communication” is transitive, i.e., if iii communicates with jjj and jjj communicates with kkk, then iii communicates with kkk.

Communication is an equivalence relation, subdividing all states into communication classes. Communication classes can be found for example by drawing transition graphs.

Lastly, a Markov chain is irreducible if it has exactly one communication class.

Definition: Recurrence and Transience

Let TjT_jTj​ be the first passage time to state jjj,

Note: Communication Classes

The states of a communication class are either all recurrent or all transient. The states of a finite irreducible Markov chain are all recurrent.

If a state is recurrent, only states inside its communication class are accessible from it. If no states outside a finite communication class are accessible from it, then the class consists of recurrent states.
