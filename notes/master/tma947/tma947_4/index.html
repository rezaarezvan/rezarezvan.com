Before we start with local and global optimality, we will have a short recap of some concepts that we need to use.

Open, closed, and bounded sets

Recall the following properties of sets in Rn\mathbb{R}^nRn,

Note: Set properties

A set S⊆RnS \subseteq \mathbb{R}^nS⊆Rn is called open if ∀x∈S, ∃ϵ>0:Bϵ(x)⊆S\forall \mathbf{x} \in S, \ \exists \epsilon > 0 : B_{\epsilon}(\mathbf{x}) \subseteq S∀x∈S, ∃ϵ>0:Bϵ​(x)⊆S.

A set S⊆RnS \subseteq \mathbb{R}^nS⊆Rn is called closed if its complement Rn∖S\mathbb{R}^n \setminus SRn∖S is open (complement of an open set).

A set S⊆RnS \subseteq \mathbb{R}^nS⊆Rn is called bounded if ∃M>0:∀x∈S:∥x∥<M, ∀x∈S\exists M > 0 : \forall \mathbf{x} \in S : \Vert \mathbf{x} \Vert < M, \ \forall \mathbf{x} \in S∃M>0:∀x∈S:∥x∥<M, ∀x∈S.

A set S⊆RnS \subseteq \mathbb{R}^nS⊆Rn is called compact if it is closed and bounded.

Local & Global Optimality

Recall, our model problem can be written as,

(P){min⁡x∈Rnf(x)subject to x∈S,\begin{align*} (P) \quad & \begin{cases} \underset{\mathbf{x} \in \mathbb{R}^n}{\min} & f(\mathbf{x}) \newline \text{subject to } & \mathbf{x} \in S, \end{cases} \end{align*}(P)​{x∈Rnmin​subject to ​f(x)x∈S,​​

Optimality conditions will characterize which points x\mathbf{x}x that can be locally/globally optimal to our problem (P)(P)(P).

So the our first question is, where can the problem have local and global solutions?

From Figure 1 (and intuition calculus), we can see that the candidates for local and global optimality are:

  • Points where the gradient is zero (stationary points).
  • Points where the gradient is undefined (nondifferentiable points).
  • Boundary points of the feasible set SSS.
  • Discontinuous points of the function fff.

Now, let’s (formally) define what we mean with global and local optimal solutions?

Definition: Global and local optimality
f(x⋆)≤f(x),∀x∈S.f(\mathbf{x}^{\star}) \leq f(\mathbf{x}), \quad \forall \mathbf{x} \in S.f(x⋆)≤f(x),∀x∈S.

x⋆∈S\mathbf{x}^{\star} \in Sx⋆∈S is a local minimum of fff over SSS if,

∃ϵ>0:f(x⋆)≤f(x),∀x∈S∩Bϵ(x⋆).\exists \epsilon > 0 : f(\mathbf{x}^{\star}) \leq f(\mathbf{x}), \quad \forall \mathbf{x} \in S \cap B_{\epsilon}(\mathbf{x}^{\star}).∃ϵ>0:f(x⋆)≤f(x),∀x∈S∩Bϵ​(x⋆).

x⋆∈S\mathbf{x}^{\star} \in Sx⋆∈S is a strict local minimum of fff over SSS if f(x⋆)<f(x)f(\mathbf{x}^{\star}) < f(\mathbf{x})f(x⋆)<f(x) holds above for x≠x⋆\mathbf{x} \neq \mathbf{x}^{\star}x=x⋆.

With these definitions, we can define the fundamental theorem of global optimality.

Theorem: Fundamental theorem of global optimality

Consider the problem

(P){min⁡x∈Rnf(x)subject to x∈S,\begin{align*} (P) \quad & \begin{cases} \underset{\mathbf{x} \in \mathbb{R}^n}{\min} & f(\mathbf{x}) \newline \text{subject to } & \mathbf{x} \in S, \end{cases} \end{align*}(P)​{x∈Rnmin​subject to ​f(x)x∈S,​​

where SSS is a convex set and fff is convex on SSS. Then, every local minimum of (P)(P)(P) is also a global minimum.

Let’s now prove this theorem.

Proof: Proof of the fundamental theorem of global optimality

We will use proof by contradiction. Let x⋆∈S\mathbf{x}^{\star} \in Sx⋆∈S be a local optimal but not a global optimal solution.

Further, let xˉ∈S\bar{\mathbf{x}} \in Sxˉ∈S be a point such that f(xˉ)<f(x⋆)f(\bar{\mathbf{x}}) < f(\mathbf{x}^{\star})f(xˉ)<f(x⋆). For any λ∈(0,1)\lambda \in (0, 1)λ∈(0,1),

Therefore,

f(λxˉ+(1−λ)x⋆)≤λf(xˉ)⏟<f(x⋆)+(1−λ)f(x⋆)<λf(x⋆)+(1−λ)f(x⋆)<f(x⋆)\begin{align*} f(\lambda \bar{\mathbf{x}} + (1 - \lambda) \mathbf{x}^{\star}) & \leq \lambda \underbrace{f(\bar{\mathbf{x}})}_{< f(\mathbf{x}^{\star})} + (1 - \lambda) f(\mathbf{x}^{\star}) \newline & < \lambda f(\mathbf{x}^{\star}) + (1 - \lambda) f(\mathbf{x}^{\star}) \newline & < f(\mathbf{x}^{\star}) \end{align*}f(λxˉ+(1−λ)x⋆)​≤λ<f(x⋆)f(xˉ)​​+(1−λ)f(x⋆)<λf(x⋆)+(1−λ)f(x⋆)<f(x⋆)​

However, if we now let λ→0\lambda \rightarrow 0λ→0, leads to a contradiction to the local optimality of x⋆\mathbf{x}^{\star}x⋆.

Definition: Weakly coercive functions

With these examples, we will now define Weierstrass’ theorem, which states the existence of globally optimal solutions.

Theorem: Weierstrass’ theorem

Consider the problem,

(P){min⁡x∈Rnf(x)subject to x∈S,\begin{align*} (P) \quad & \begin{cases} \underset{\mathbf{x} \in \mathbb{R}^n}{\min} & f(\mathbf{x}) \newline \text{subject to } & \mathbf{x} \in S, \end{cases} \end{align*}(P)​{x∈Rnmin​subject to ​f(x)x∈S,​​

and assume that,

  1. SSS is a non-empty and closed set,
  2. fff is continuous on SSS 1Technically, this still holds for fff is lower semi-continuous on SSS, but we will not cover that here.,
  3. fff is weakly coercive with respect to SSS.

Then, there exists a non-empty, compact set of global minimizers to (P)(P)(P).

Before we start on optimality conditions, we will see two types of optimality conditions, necessary and sufficient conditions.

Necessary optimality conditions are of the form,

x⋆ is a local minimum   ⟹   "something" holds.\mathbf{x}^{\star} \text{ is a local minimum } \implies \text{ "something" holds.}x⋆ is a local minimum ⟹ "something" holds.

They are called necessray because, if we negate the above logical statement, we get,

"something" does not hold   ⟹  x⋆ is not a local minimum.\text{"something" does not hold } \implies \mathbf{x}^{\star} \text{ is not a local minimum.}"something" does not hold ⟹x⋆ is not a local minimum.

So, “something” is a property that x\mathbf{x}x must have in order to (potentially) be a local minimum.

The other type is sufficient optimality conditions, which are of the form,

"something" holds   ⟹  x⋆ is a local minimum.\text{"something" holds } \implies \mathbf{x}^{\star} \text{ is a local minimum.}"something" holds ⟹x⋆ is a local minimum.

They are called sufficient because, if “something” holds, then we are guaranteed that x\mathbf{x}x is a local minimum. Sufficient conditions are stronger and more nice, but they are also harder to come by.

Optimality conditions for constrained problems (S⊆RnS \subseteq \mathbb{R}^nS⊆Rn)

We will now look at optimality conditions for constrained problems. When we dealt with unconstrained problems, we used the fact that we could move in any direction.

But this is not the case for constrained problems, we can only move in directions that keeps us in the feasible set SSS.

Let’s define what a feasible and descent direction is.

Definition: Feasible direction

Let x∈S\mathbf{x} \in Sx∈S, a vector p∈Rn\mathbf{p} \in \mathbb{R}^np∈Rn is called a feasible direction at the point x\mathbf{x}x, if,

∃δ>0:x+αp∈S,∀α∈[0,δ]\exists \delta > 0 : \mathbf{x} + \alpha \mathbf{p} \in S, \quad \forall \alpha \in [0, \delta]∃δ>0:x+αp∈S,∀α∈[0,δ]
Definition: Descent direction

Let x∈S\mathbf{x} \in Sx∈S, a vector p∈Rn\mathbf{p} \in \mathbb{R}^np∈Rn is called a descent direction with respect to fff at the point x\mathbf{x}x, if,

∃δ>0:f(x+αp)<f(x),∀α∈(0,δ]\exists \delta > 0 : f(\mathbf{x} + \alpha \mathbf{p}) < f(\mathbf{x}), \quad \forall \alpha \in (0, \delta]∃δ>0:f(x+αp)<f(x),∀α∈(0,δ]
Note
  • For x⋆\mathbf{x}^{\star}x⋆ to be a local minimum, there cannot be any feasible descent directions at x⋆\mathbf{x}^{\star}x⋆.
  • Recall that, for two vectors, y,z∈Rn\mathbf{y}, \mathbf{z} \in \mathbb{R}^ny,z∈Rn, we have that,
yTz=∥y∥∥z∥cos⁡(θ),\mathbf{y}^T \mathbf{z} = \Vert \mathbf{y} \Vert \Vert \mathbf{z} \Vert \cos(\theta),yTz=∥y∥∥z∥cos(θ),
