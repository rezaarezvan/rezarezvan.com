Convex Sets

Convex Hull

Definition: Convex Hull

The convex hull of a finite set of points, {v1,…,vk}⊆Rn\{\mathbf{v}_1, \ldots, \mathbf{v}_k \} \subseteq \mathbb{R}^n{v1​,…,vk​}⊆Rn is defined as,

conv V≔{λ1v1+…+λkvk∣λ1,…,λk≥0∑i=1kλi=1}\mathrm{conv} \ V \coloneqq \{\lambda_1 \mathbf{v}_1 + \ldots + \lambda_k \mathbf{v}_k \mid \lambda_1, \ldots, \lambda_k \geq 0 \sum_{i=1}^k \lambda_i = 1 \}conv V:={λ1​v1​+…+λk​vk​∣λ1​,…,λk​≥0i=1∑k​λi​=1}
Definition: Properties of the Convex Hull

The convex hull of the set S⊆RnS \subseteq \mathbb{R}^nS⊆Rn, is the set with any of the following properties,

  • (1) It is the unique minimal convex set containing SSS.
  • (2) It is the intersection of all convex sets containing SSS.
  • (3) It is the set of all convex combinations of points in SSS.

From (3), any point in the convex hull can be expressed as a convex combination of points in SSS.

An interesting question is how many points do we need to construct the convex hull?

Theorem: Carathéodory’s Theorem

Let α∈conv(S)\alpha \in \mathrm{conv}(S)α∈conv(S), where S⊆RnS \subseteq \mathbb{R}^nS⊆Rn.

Then α\alphaα can be expressed as a convex combination of at most n+1n+1n+1 points in SSS.

Polytopes

Definition: Polytope

A set P⊆RnP \subseteq \mathbb{R}^nP⊆Rn is called a polytope if it is the convex hull of finitely many points.

Note

A polytope will always have straight edges. So, a circle is not a polytope (since the circumference is curved and can not be expressed with finitely many points).

Definition: Extreme Point

A point v\mathbf{v}v of a convex set SSS is called an extreme point if,

Theorem: Extreme Points of a Polytope

Let PPP be the polytope conv(V)\mathrm{conv}(V)conv(V), where V={v1,…,vk}V = \{\mathbf{v}_1, \ldots, \mathbf{v}_k \}V={v1​,…,vk​}. Then, PPP is equal to the convex hull of its extreme points.

Representation Theorem

Theorem: Representation Theorem

Let Q={x∈Rm∣Ax≤b}Q = \{\mathbf{x} \in \mathbb{R}^m \mid A\mathbf{x} \leq \mathbf{b} \}Q={x∈Rm∣Ax≤b} (polyhedron) and let {v1,…,vk}\{\mathbf{v}_1, \ldots, \mathbf{v}_k \}{v1​,…,vk​} be its extreme points. Further, we define P≔conv({v1,…,vk})P \coloneqq \mathrm{conv}(\{\mathbf{v}_1, \ldots, \mathbf{v}_k \})P:=conv({v1​,…,vk​}) (polytope) and C≔{x∈Rm∣Ax≤0}C \coloneqq \{\mathbf{x} \in \mathbb{R}^m \mid A\mathbf{x} \leq 0 \}C:={x∈Rm∣Ax≤0} (cone). Then,

Q=P+C={x∈Rm∣x=u+v, u∈P, v∈C}\begin{align*} Q & = P + C \newline & = \{\mathbf{x} \in \mathbb{R}^m \mid \mathbf{x} = \mathbf{u} + \mathbf{v}, \ \mathbf{u} \in P, \ \mathbf{v} \in C \} \end{align*}Q​=P+C={x∈Rm∣x=u+v, u∈P, v∈C}​
Corollary: Bounded Polyhedron

A bounded polyhedron is a polytope.

Let’s state the pros and cons of polytopes and polyhedra.

  • Polytope
    • Pros: Easy to get extreme points (by definition).
    • Cons: Hard to check if new point is inside.
  • Polyhedron
    • Pros: Easy to check if new point is inside.
    • Cons: Hard to get extreme points (need to check intersection of half-spaces, grows combinatorially, infeasible for large problems).
