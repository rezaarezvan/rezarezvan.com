Constraint Qualifications

We will now introduce some more constraint qualifications that are easier to check than Abadie’s constraint qualifications, but imply that Abadie’s constraint qualifications hold.

Definition: Linear Independence Constraint Qualification (LICQ)

The Linear Independence Constraint Qualification (LICQ) holds at a point x⋆\mathbf{x}^{\star}x⋆ if ∇gi(x⋆),i∈I(x⋆)\nabla g_i(\mathbf{x}^{\star}), i \in \mathcal{I}(\mathbf{x}^{\star})∇gi​(x⋆),i∈I(x⋆) are linearly independent.

Theorem
LICQ  ⟹  Abadie’s CQ\text{LICQ} \implies \text{Abadie's CQ}LICQ⟹Abadie’s CQ
Definition: Affine Independence Constraint Qualification (Affine CQ)

The Affine Independence Constraint Qualification (Affine CQ) holds if all gi(x),i∈I(x⋆)g_i(\mathbf{x}), i \in \mathcal{I}(\mathbf{x}^{\star})gi​(x),i∈I(x⋆) are affine functions and ∇gi(x⋆),i=1,…,m\nabla g_i(\mathbf{x}^{\star}), i = 1, \ldots, m∇gi​(x⋆),i=1,…,m are affine.

Theorem
Affine CQ  ⟹  Abadie’s CQ. (In all points x∈S)\text{Affine CQ} \implies \text{Abadie's CQ. (In all points } \mathbf{x} \in S\text{)}Affine CQ⟹Abadie’s CQ. (In all points x∈S)
Definition: Slater’s Constraint Qualification (Slater’s CQ)

Slater’s Constraint Qualification (Slater’s CQ) holds if all gi(x),i=1,…,mg_i(\mathbf{x}), i = 1, \ldots, mgi​(x),i=1,…,m are convex functions and if an interior point exists, i.e., ∃ x0 such that gi(x0)<0,i=1,…,m\exists \ \mathbf{x}_0 \ \text{such that} \ g_i(\mathbf{x}_0) < 0, i = 1, \ldots, m∃ x0​ such that gi​(x0​)<0,i=1,…,m.

Theorem
Slater’s CQ  ⟹  Abadie’s CQ. (In all points x∈S)\text{Slater's CQ} \implies \text{Abadie's CQ. (In all points } \mathbf{x} \in S\text{)}Slater’s CQ⟹Abadie’s CQ. (In all points x∈S)

Karush-Kuhn-Tucker (KKT) Conditions with Equality Constraints

Connection to Lagrangian Relaxation

We can also connect the KKT conditions to the Lagrangian relaxation.
