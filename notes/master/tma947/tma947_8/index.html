Introduction

In this part we will cover the concept of Lagrangian duality, which is a powerful tool in optimization theory. We will introduce, discuss and prove several important results, including weak and strong duality.

We will then connect this to global optimality conditions, and show how duality can be used to derive optimality conditions for (un)constrained optimization problems. Finally, we will also (very shortly) introduce Linear Programming (LP) and prove Farkas’ lemma using duality.

The Lagrangian: Dual Problem

Definition 3 (Lagrangian Dual Problem)

One can understand that the dual function q(μ)q(\boldsymbol{\mu})q(μ) gives a lower bound on the optimal value of (P)(P)(P) for any μ≥0\boldsymbol{\mu} \geq \mathbf{0}μ≥0. Thus, we can try to find the “best” lower bound by solving the dual problem, i.e., maximizing q(μ)q(\boldsymbol{\mu})q(μ) over μ≥0\boldsymbol{\mu} \geq \mathbf{0}μ≥0,

The (Lagrangian) dual problem is defined as,

(D){sup⁡ q(μ)subject to μ≥0\begin{align*} (D) \quad & \begin{cases} \sup \ & q(\boldsymbol{\mu}) \newline \text{subject to} \ & \boldsymbol{\mu} \geq \mathbf{0} \end{cases} \end{align*}(D)​{sup subject to ​q(μ)μ≥0​​
Note

From weak duality, if q⋆q^{\star}q⋆ is the optimal value of (D)(D)(D), then q⋆≤f⋆q^{\star} \leq f^{\star}q⋆≤f⋆, i.e., the optimal value of the dual problem is a lower bound on the optimal value of the primal problem. This is also what we saw in the last part with the relaxtion theorem.

Theorem 2 (Convexity of the Dual Function)

The dual function qqq is concave and its effective domain, defined as,

{μ∈Rn∣q(μ)>−∞},\{\boldsymbol{\mu} \in \mathbb{R}^n \mid q(\boldsymbol{\mu}) > -\infty\},{μ∈Rn∣q(μ)>−∞},

is a convex set. Thus (D)(D)(D) is a convex optimization problem, even if (P)(P)(P) is not.

Note

In the dual problem, we are maximizing a concave function on a convex set, which is equivalent to minimizing a convex function on a convex set, thus (D)(D)(D) is a convex optimization problem.

Linear Programming and Farkas’ Lemma

We will now (very shortly) introduce Linear Programming (LP) and prove Farkas’ lemma using duality.

Definition 5 (Linear Programming)

A linear progrm (LP) on standard form is defined as,

(P){inf⁡ cTxsubject to Ax=bx≥0\begin{align*} (P) \quad & \begin{cases} \inf \ & \mathbf{c}^T \mathbf{x} \newline \text{subject to} \ & A \mathbf{x} = \mathbf{b} \newline & \mathbf{x} \geq \mathbf{0} \end{cases} \end{align*}(P)​⎩⎨⎧​inf subject to ​cTxAx=bx≥0​​

Thus, the Lagrangian for a LP is,

L(x,y)=cTx+yT(b−Ax).\mathcal{L}(\mathbf{x}, \mathbf{y}) = \mathbf{c}^T \mathbf{x} + \mathbf{y}^T (b - A \mathbf{x}).L(x,y)=cTx+yT(b−Ax).

With this, we can now prove Farkas’ lemma in a very elegant way using duality.

Theorem 5 (Farkas’ Lemma)

Let A∈Rm×nA \in \mathbb{R}^{m \times n}A∈Rm×n and b∈Rm\mathbf{b} \in \mathbb{R}^mb∈Rm. Then, exactly one of the following systems is solvable,

{Ax=bx≥0{ATy≤0bTy>0\begin{cases} A \mathbf{x} = \mathbf{b} \newline \mathbf{x} \geq 0 \end{cases} \quad \quad \begin{cases} A^T \mathbf{y} \leq 0 \newline \mathbf{b}^T \mathbf{y} > 0 \end{cases}{Ax=bx≥0​{ATy≤0bTy>0​
Proof (Farkas’ Lemma)

If the first system has a solution x\mathbf{x}x, then,

bTy=xTATy≤0,\mathbf{b}^T \mathbf{y} = \mathbf{x}^T A^T \mathbf{y} \leq 0,bTy=xTATy≤0,

But x≥0\mathbf{x} \geq 0x≥0 so ATy≤0A^T \mathbf{y} \leq 0ATy≤0 cannot hold, which means that the second system is infeasible.

Assume that the second system is infeasible. Consider the LP,

min⁡ 0Txsubject to Ax=b x≥0\begin{align*} \min \ & \mathbf{0}^T \mathbf{x} \newline \text{subject to} \ & A \mathbf{x} = \mathbf{b} \newline \ & \mathbf{x} \geq 0 \end{align*}min subject to  ​0TxAx=bx≥0​

and its dual program,

max⁡ bTysubject to ATy≤0 y free\begin{align*} \max \ & \mathbf{b}^T \mathbf{y} \newline \text{subject to} \ & A^T \mathbf{y} \leq 0 \newline \ & \mathbf{y} \text{ free} \end{align*}max subject to  ​bTyATy≤0y free​

Since the second system is infeasible, y=0\mathbf{y} = \mathbf{0}y=0 is an optimal solution to the primal problem with optimal value 000. Hence the Strong Duality theorem implies that there exists an optimal solution x⋆\mathbf{x}^{\star}x⋆ to the dual problem with optimal value 000. This solution is feasible to the first system.

What we have proved above is the equivalence,

(I)  ⟺  ¬(II)(I) \iff \neg (II)(I)⟺¬(II)

Logically, this is equivalent to the statement that,

¬(I)  ⟺  (II)\neg (I) \iff (II)¬(I)⟺(II)

We have hence established that precisely one of the systems is solvable. ■_\blacksquare■​
