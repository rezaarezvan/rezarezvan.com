Introduction

In this part we will cover the Simplex method, which is a very efficient method for solving Linear Programs (LPs) in practice.

Firstly, let’s recap the setting we are working with.

Recall (Linear Program on Standard Form)

An LP on standard form is,

min⁡ cTxsubject to Ax=b x≥0\begin{align*} \min \ & \mathbf{c}^T \mathbf{x} \newline \text{subject to} \ & A \mathbf{x} = \mathbf{b} \newline \ & \mathbf{x} \geq \mathbf{0} \end{align*}min subject to  ​cTxAx=bx≥0​

where A∈Rm×nA \in \mathbb{R}^{m \times n}A∈Rm×n, b∈Rm\mathbf{b} \in \mathbb{R}^mb∈Rm, c∈Rn\mathbf{c} \in \mathbb{R}^nc∈Rn and x∈Rn\mathbf{x} \in \mathbb{R}^nx∈Rn.

Note

We will also assume that rank(A)=m,m≤n\mathrm{rank}(A) = m, m \leq nrank(A)=m,m≤n and b≥0\mathbf{b} \geq \mathbf{0}b≥0.

Recall (Basic Feasible Solution)

Recall the standard form polyhedron P={x∈Rn∣Ax=b, x≥0}P = \{\mathbf{x} \in \mathbb{R}^n \mid A \mathbf{x} = \mathbf{b}, \ \mathbf{x} \geq \mathbf{0}\}P={x∈Rn∣Ax=b, x≥0}. A point xˉ\bar{\mathbf{x}}xˉ is called a basic feasible solution (BFS), if,

  1. xˉ\bar{\mathbf{x}}xˉ is a basic solution, i.e., Axˉ=bA \bar{\mathbf{x}} = \mathbf{b}Axˉ=b and the columns of AAA corresponding to the non-zero components of xˉ\bar{\mathbf{x}}xˉ are linearly independent,
  2. xˉ≥0\bar{\mathbf{x}} \geq \mathbf{0}xˉ≥0
Note

A BFS is a point in the feasible region (polyhedron) PPP.

The Simplex Method

We will now introduce the Simplex method, which consists of four steps.

Algorithm (The Simplex Method)
  1. Determine if the current BFS is optimal. If it is, stop.
  2. If it is not optimal, find a search direction d\mathbf{d}d to an adjacent BFS that decreases the objective function (adjacent and descent).
  3. Determine the step length θ\thetaθ to the adjacent BFS in the direction d\mathbf{d}d.
  4. Update to the new BFS x+θd\mathbf{x} + \theta \mathbf{d}x+θd and go to step 1.

We will now go through each step in detail, however, we will go in the order (2), (1), (3), (4), you will see why soon.

Definition 2 (Reduced Cost)

Let xˉ\bar{\mathbf{x}}xˉ be a BFS corresponding to the partitioning A=[BN]A = \begin{bmatrix} B & N \end{bmatrix}A=[B​N​], the reduced costs for this partitioning is defined as,

Let’s now define the optimality condition.

Theorem 1 (Optimality Check)

Let xˉ\bar{\mathbf{x}}xˉ be a BFS corresponding to the partitioning A=[BN]A = \begin{bmatrix} B & N \end{bmatrix}A=[B​N​], and cN~T=cNT−cBTB−1N\tilde{\mathbf{c}_N}^T = \mathbf{c}_N^T - \mathbf{c}_B^T B^{-1} NcN​~​T=cNT​−cBT​B−1N be the reduced costs for this partitioning,

cN~T≥0  ⟹  xˉ is optimal\tilde{\mathbf{c}_N}^T \geq \mathbf{0} \implies \bar{\mathbf{x}} \text{ is optimal}cN​~​T≥0⟹xˉ is optimal
Note

G> > The above statement is a sufficient condition for optimality, which is nice.

Further, note that, if we do not have any degeneracy, then the above statement is a necessary and sufficient condition for optimality, i.e.,

cN~T≥0  ⟺  xˉ is optimal (no degeneracy)\tilde{\mathbf{c}_N}^T \geq \mathbf{0} \iff \bar{\mathbf{x}} \text{ is optimal (no degeneracy)}cN​~​T≥0⟺xˉ is optimal (no degeneracy)

Step 3: Determining the Step Length

We now have a search direction d=[−B−1Nejej]\mathbf{d} = \begin{bmatrix} -B^{-1} N \mathbf{e}_j \newline \mathbf{e}_j \end{bmatrix}d=[−B−1Nej​ej​​], and we want to determine the step length θ\thetaθ such that xˉ+θd\bar{\mathbf{x}} + \theta \mathbf{d}xˉ+θd is an adjacent BFS.

Let’s write out the update,

xˉ+θd=[xˉBxˉN]+θ[−B−1Nejej]=[B−1b0]+θ[−B−1Nejej]=[B−1b−θB−1Nej0+θej]\begin{align*} \bar{\mathbf{x}} + \theta \mathbf{d} & = \begin{bmatrix} \bar{\mathbf{x}}_B \newline \bar{\mathbf{x}}_N \end{bmatrix} + \theta \begin{bmatrix} -B^{-1} N \mathbf{e}_j \newline \mathbf{e}_j \end{bmatrix} \newline & = \begin{bmatrix} B^{-1} \mathbf{b} \newline \mathbf{0} \end{bmatrix} + \theta \begin{bmatrix} -B^{-1} N \mathbf{e}_j \newline \mathbf{e}_j \end{bmatrix} \newline & = \begin{bmatrix} B^{-1} \mathbf{b} - \theta B^{-1} N \mathbf{e}_j \newline \mathbf{0} + \theta \mathbf{e}_j \end{bmatrix} \newline \end{align*}xˉ+θd​=[xˉB​xˉN​​]+θ[−B−1Nej​ej​​]=[B−1b0​]+θ[−B−1Nej​ej​​]=[B−1b−θB−1Nej​0+θej​​]​

Recall that we want x≥0\mathbf{x} \geq \mathbf{0}x≥0, thus we need to ensure that both xˉB−θB−1Nej≥0\bar{\mathbf{x}}_B - \theta B^{-1} N \mathbf{e}_j \geq \mathbf{0}xˉB​−θB−1Nej​≥0 and xˉN+θej≥0\bar{\mathbf{x}}_N + \theta \mathbf{e}_j \geq \mathbf{0}xˉN​+θej​≥0.

The first condition however can cause problems, if −B−1Nej≤0-B^{-1} N \mathbf{e}_j \leq \mathbf{0}−B−1Nej​≤0, then we can increase θ\thetaθ indefinitely and still satisfy the condition, i.e., the LP is unbounded!

If −B−1Nej≰0-B^{-1} N \mathbf{e}_j \not\leq \mathbf{0}−B−1Nej​≤0, then we can find at least one component iii such that (−B−1Nej)i>0(-B^{-1} N \mathbf{e}_j)_i > 0(−B−1Nej​)i​>0, and thus we can increase θ\thetaθ until the iii-th component of xˉB−θB−1Nej\bar{\mathbf{x}}_B - \theta B^{-1} N \mathbf{e}_jxˉB​−θB−1Nej​ is zero.

Note

We note that the index kkk that attains the minimum is interesting. This index kkk tells us which basic variable will become non-basic in the next iteration, i.e., which column of BBB will be replaced by the jjj-th column of NNN.

Step 4: Update to the New BFS

We now have everything we need to perform the update to the new BFS.

Definition 4 (Simplex Update)

Assume that we have a search direction d\mathbf{d}d from (2) and a step length θ⋆\theta^{\star}θ⋆ from (3), then the new BFS is,

xˉnew=xˉ+θ⋆d\bar{\mathbf{x}}^{\text{new}} = \bar{\mathbf{x}} + \theta^{\star} \mathbf{d}xˉnew=xˉ+θ⋆d

Similarly, we can update the basis matrix BBB by,

B~−1b,\tilde{B}^{-1} b,B~−1b,

where B~\tilde{B}B~ is the basis matrix obtained by replacing the kkk-th column of BBB with the jjj-th column of NNN, where kkk is the index that attains the minimum in the minimum ratio test.

Let’s now formally write out the Simplex method.

Algorithm (The Simplex Method)
  1. Let xˉ\bar{\mathbf{x}}xˉ be a initial BFS, corresponding to the partitioning A=[BN]A = \begin{bmatrix} B & N \end{bmatrix}A=[B​N​].
  2. Compute the reduced costs cN~T=cNT−cBTB−1N\tilde{\mathbf{c}_N}^T = \mathbf{c}_N^T - \mathbf{c}_B^T B^{-1} NcN​~​T=cNT​−cBT​B−1N.
  • If cN~T≥0\tilde{\mathbf{c}_N}^T \geq \mathbf{0}cN​~​T≥0, stop, xˉ\bar{\mathbf{x}}xˉ is optimal.
  • Else, choose j∈arg⁡min⁡k:(cN~)k<0(cN~)kj \in \underset{k : (\tilde{\mathbf{c}_N})_k < 0}{\arg \min} (\tilde{\mathbf{c}_N})_kj∈k:(cN​~​)k​<0argmin​(cN​~​)k​.
  1. Compute the search direction d=[−B−1Nejej]\mathbf{d} = \begin{bmatrix} -B^{-1} N \mathbf{e}_j \newline \mathbf{e}_j \end{bmatrix}d=[−B−1Nej​ej​​].
  • If −B−1Nej≤0-B^{-1} N \mathbf{e}_j \leq \mathbf{0}−B−1Nej​≤0, stop, the LP is unbounded.
  1. Compute the step length θ⋆=min⁡k:(B−1Nej)k>0(B−1b)k(B−1Nej)k\theta^{\star} = \underset{k : (B^{-1} N \mathbf{e}_j)_k > 0}{\min} \frac{(B^{-1} \mathbf{b})_k}{(B^{-1} N \mathbf{e}_j)_k}θ⋆=k:(B−1Nej​)k​>0min​(B−1Nej​)k​(B−1b)k​​.
  2. Update to the new BFS xˉnew=xˉ+θ⋆d\bar{\mathbf{x}}^{\text{new}} = \bar{\mathbf{x}} + \theta^{\star} \mathbf{d}xˉnew=xˉ+θ⋆d and update the basis matrix BBB by replacing the kkk-th column of BBB with the jjj-th column of NNN, where kkk is the index that attains the minimum in the minimum ratio test.
  3. Repeat from step 2.

We will now state a theorem regarding the convergence of the Simplex method and prove it.

Theorem 2 (Finite Termination of the Simplex Method)

If PPP is nonempty and there is no degeneracy, then the Simplex method terminates in a finite number of iterations with an optimal solution or with the conclusion that the problem is unbounded.

Proof

If a BFS is non-degenerate, then it has exactly mmm positive components, and hence a unique basis matrix. We know that if the search direction is unbounded, then the LP is unbounded and terminates. Otherwise, the basis is either optimal or θ⋆>0\theta^{\star} > 0θ⋆>0.

Therefore, in each iteration, the objective function is strictly decreased. Hence, no BFS can be revisited, and since there is a finite number of BFSs, the method must terminate in a finite number of iterations.

Note

Remember that there are (nm)\binom{n}{m}(mn​) possible basis matrices, thus at most (nm)\binom{n}{m}(mn​) iterations, i.e., the worst-case complexity is exponential in nnn and mmm.

Initial BFS and Phase-I problem

We have now covered the Simplex method, however, we have not discussed how to find an initial BFS.

If we are lucky and have an LP of the form,

min⁡ cxTx+cyTysubject to Ax+Iy=b x,y≥0\begin{align*} \min \ & \mathbf{c}_x^T \mathbf{x} + \mathbf{c}_y^T \mathbf{y} \newline \text{subject to} \ & A \mathbf{x} + I \mathbf{y} = \mathbf{b} \newline \ & \mathbf{x}, \mathbf{y} \geq \mathbf{0} \end{align*}min subject to  ​cxT​x+cyT​yAx+Iy=bx,y≥0​

We can let (x=0,y=b)T(\mathbf{x} = \mathbf{0}, \mathbf{y} = \mathbf{b})^T(x=0,y=b)T be our initial BFS, with basis matrix B=IB = IB=I.

However, in general, we will not be this lucky, but we can construct an auxiliary LP, called the Phase-I problem, to find an initial BFS.

Definition 5 (Phase-I Problem)

Given an LP on standard form, we can construct the Phase-I problem,

min⁡ 1Tysubject to Ax+Iy=b x,y≥0\begin{align*} \min \ & \mathbf{1}^T \mathbf{y} \newline \text{subject to} \ & A \mathbf{x} + I \mathbf{y} = \mathbf{b} \newline \ & \mathbf{x}, \mathbf{y} \geq \mathbf{0} \end{align*}min subject to  ​1TyAx+Iy=bx,y≥0​

where 1=[11…1]T∈Rm\mathbf{1} = \begin{bmatrix} 1 & 1 & \ldots & 1 \end{bmatrix}^T \in \mathbb{R}^m1=[1​1​…​1​]T∈Rm and y∈Rm\mathbf{y} \in \mathbb{R}^my∈Rm are artificial variables.

Note

The Phase-I problem always has an initial BFS, namely (x=0,y=b)T(\mathbf{x} = \mathbf{0}, \mathbf{y} = \mathbf{b})^T(x=0,y=b)T, with basis matrix B=IB = IB=I.

Further, the Phase-I problem is always feasible.

  1. If the optimal value of the Phase-I problem is zero, then the original LP is feasible,

  2. If the optimal value of the Phase-I problem is positive, then the original LP is infeasible.
